# ชีทสูตรล้วนฟิสิกส์ A-Level 69

## กลศาสตร์

- พื้นที่ใต้กราฟ $v-t$ = การกระจัด
- ความชันกราฟ $v-t$ = ความเร่ง
- $u_x = u\cos\theta$
- $u_y = u\sin\theta$
- $t = 2u_y/g$
- $f_k = \mu_k N$
- $W_{net} = \Delta K$
- $U_s = \frac{1}{2}kx^2$
- $U_g = mgh$
- $p = mv$
- $p_{before} = p_{after}$
- $x = A\sin(\omega t + \phi)$
- $v = \omega A\cos(\omega t + \phi)$
- $F_c = mv^2/r$
- ถ้าสปริงเป็นแรงเข้าสู่ศูนย์กลาง: $kx = mv^2/r$

## คลื่น เสียง แสง

- $v = f\lambda$
- $T = 1/f$
- ท่อปลายปิด: $L = (2n-1)\lambda/4$
- $\Delta\beta = 10\log(I_2/I_1)$
- เลนส์บาง: $1/f = 1/s + 1/s'$ 
- ดอปเพลอร์: เข้า = ความถี่สูงขึ้น, ออก = ความถี่ต่ำลง
- เกรตติง: $d\sin\theta = n\lambda$
- ตำแหน่งบนฉาก: $x \approx L\tan\theta$

## ไฟฟ้าและแม่เหล็ก

- $V = IR$
- $I = \varepsilon/(R+r)$
- $V = kQ/r$
- $W = q\Delta V$
- $\Phi = BA\cos\theta$
- $\varepsilon = |\Delta\Phi/\Delta t|$
- $qvB = mv^2/r$
- $I = neAv_d$
- $Q = CV$
- $U = Q^2/(2C) = \frac{1}{2}CV^2$
- $Z = \sqrt{R^2 + (X_L - X_C)^2}$

## แก๊ส ของไหล ของแข็ง

- $PV = nRT$
- $v_{rms} = \sqrt{3kT/m}$ หรือ $\sqrt{3RT/M}$
- $F_b = \rho gV$
- $Y = FL/(A\Delta L)$
- $\Delta U = Q - W$

## ฟิสิกส์สมัยใหม่

- โฟโตอิเล็กทริก: $hf = W + K_{max}$
- เดอบรอยล์: $\lambda = h/p = h/\sqrt{2mK}$
- กัมมันตภาพ: $N = N_0(1/2)^{t/T}$
- กัมมันตภาพ: $A = \lambda N$
- โบร์: $E_n = -13.6/n^2$ eV
- พลังงานโฟตอน: $\Delta E = hc/\lambda$

## จำสั้นมาก

- กราฟ = ดูพื้นที่และความชัน
- ตัวแปร = จัดรูปก่อนแทนเลข
- หน่วย = แปลงเป็น SI เสมอ
- อัตราส่วน = ตัดค่าคงที่ก่อน
- สองบทผสม = หา "ตัวแปรเชื่อม"
