# เนื้อหาเพื่อใช้เรียนเตรียมสอบฟิสิกส์ A-Level 69

เอกสารนี้ทำไว้สำหรับอ่านทบทวนก่อนสอบ โดยเน้น "เข้าใจโจทย์ -> เลือกสูตร -> จัดรูป -> เช็คหน่วย" มากกว่าท่องอย่างเดียว

## วิธีใช้เอกสารนี้

- อ่านภาพรวมของบทก่อน
- จำสูตรแกน 1-2 สูตรต่อบท
- ทำโจทย์ตัวอย่างสั้นๆ เพื่อเช็คว่าพอเอาไปใช้ได้จริง
- ถ้าบทไหนพลาดบ่อย ให้กลับไปอ่านข้อสอบจริงที่ตรงบทนั้นทันที

## 1. กลศาสตร์

### 1.1 การเคลื่อนที่แนวตรง

- ถ้าให้กราฟ $v-t$ ให้หาพื้นที่ใต้กราฟก่อนเสมอ เพราะคือการกระจัดหรือระยะทาง
- ความชันของกราฟ $v-t$ คือความเร่ง
- ความเร็วคงที่แปลว่า $a = 0$

### 1.2 โปรเจกไทล์

- แตกความเร็วต้นเป็น
  - $u_x = u \cos \theta$
  - $u_y = u \sin \theta$
- แนวราบไม่มีความเร่ง
- เวลาบินทั้งหมดถ้าตกที่ระดับเดิมคือ $t = 2u_y/g$

### 1.3 แรงและการเคลื่อนที่

- ใช้กฎของนิวตัน: $\Sigma F = ma$
- ถ้าวัตถุเคลื่อนที่ด้วยความเร็วคงที่ แปลว่าแรงลัพธ์เป็นศูนย์
- แรงเสียดทานจลน์คือ $f_k = \mu_k N$

### 1.4 งานและพลังงาน

- งานสุทธิเท่ากับการเปลี่ยนพลังงานจลน์: $W_{net} = \Delta K$
- พลังงานศักย์สปริง: $U_s = \frac{1}{2}kx^2$
- พลังงานศักย์โน้มถ่วง: $U_g = mgh$

### 1.5 โมเมนตัม

- โมเมนตัม: $p = mv$
- กฎอนุรักษ์โมเมนตัมใช้เมื่อแรงภายนอกสุทธิน้อยมากหรือไม่มี
- การชนใน 2 มิติ ถ้ามวลเท่ากันและชนยืดหยุ่น มักใช้เวกเตอร์หรือสามเหลี่ยม 3:4:5 ได้

### 1.6 SHM

- สมการพื้นฐาน: $x = A\sin(\omega t + \phi)$
- ความเร็ว: $v = \omega A\cos(\omega t + \phi)$
- ที่ปลายสุด ความเร็วเป็นศูนย์
- ที่ตำแหน่งสมดุล ความเร็วมากสุด

### 1.7 การเคลื่อนที่วงกลม

- แรงเข้าสู่ศูนย์กลาง: $F_c = mv^2/r$
- ถ้ามีสปริงดึงให้หมุน ใช้ $kx = mv^2/r$
- ต้องหาระยะยืดหรือรัศมีจากรูปก่อนแทนสูตร

## 2. คลื่น เสียง และแสง

### 2.1 คลื่นกล

- อัตราเร็วคลื่น: $v = f\lambda$
- คาบ: $T = 1/f$
- ถ้าคลื่นเลื่อนเป็นครึ่งความยาวคลื่น จะกินเวลาครึ่งคาบ

### 2.2 เสียง

- ท่อปลายปิดสั่นพ้องได้เฉพาะฮาร์มอนิกคี่
- ความยาวท่อปลายปิด: $L = (2n-1)\lambda/4$
- ระดับเสียงเดซิเบลใช้ลอการิทึม อย่าสับสนกับอัตราส่วนตรงๆ

### 2.3 เลนส์

- สูตรเลนส์บาง: $1/f = 1/s + 1/s'$ 
- กำหนดเครื่องหมายระยะให้ตรงชนิดภาพ
- ถ้าเป็นเลนส์หลายชิ้น ให้หาภาพจากชิ้นแรกเป็นวัตถุของชิ้นถัดไป

### 2.4 ดอปเพลอร์

- ความถี่ที่ได้ยินเปลี่ยนเพราะแหล่งกำเนิดหรือผู้ฟังเคลื่อนที่
- ทิศเข้า: ความถี่สูงขึ้น
- ทิศออก: ความถี่ต่ำลง
- จุดผิดบ่อยคือเครื่องหมายในตัวส่วน

### 2.5 เกรตติง

- สมการหลัก: $d\sin\theta = n\lambda$
- ถามระยะบนฉาก มักใช้ $x \approx L\tan\theta$
- ถ้ามุมเล็กมาก ใช้ประมาณ $\sin\theta \approx \tan\theta$

## 3. ไฟฟ้าและแม่เหล็ก

### 3.1 ไฟฟ้าสถิต

- ศักย์ไฟฟ้าจากประจุจุด: $V = kQ/r$
- งานในการย้ายประจุ: $W = q\Delta V$
- ศักย์เป็นสเกลาร์ จึงบวกจากแต่ละประจุได้ตรงๆ

### 3.2 วงจรไฟฟ้ากระแสตรง

- กฎของโอห์ม: $V = IR$
- แบตมีความต้านทานภายใน: $I = \varepsilon/(R+r)$
- ต่ออนุกรม: กระแสเท่ากัน
- ต่อขนาน: ความต่างศักย์เท่ากัน

### 3.3 กระแสและความเร็วลอยเลื่อน

- กระแส: $I = neAv_d$
- ถ้ากระแสลด ความเร็วลอยเลื่อนลดตามเมื่อ $n,e,A$ คงที่

### 3.4 อนุภาคในสนามแม่เหล็ก

- แรงแม่เหล็ก: $F = qvB$
- การเคลื่อนที่เป็นวงกลม: $qvB = mv^2/r$
- ถ้าเร่งด้วยความต่างศักย์ก่อนเข้า B: $qV = \frac{1}{2}mv^2$

### 3.5 การเหนี่ยวนำ

- แรงเคลื่อนไฟฟ้าเหนี่ยวนำ: $\varepsilon = |\Delta\Phi/\Delta t|$
- ฟลักซ์แม่เหล็ก: $\Phi = BA\cos\theta$

### 3.6 คาปาซิเตอร์

- ประจุสะสม: $Q = CV$
- พลังงานสะสม: $U = Q^2/(2C) = \frac{1}{2}CV^2$
- ต่อขนานแล้วประจุรวมคงอยู่ตามกฎอนุรักษ์ประจุ

### 3.7 AC / RLC

- อิมพีแดนซ์: $Z = \sqrt{R^2 + (X_L - X_C)^2}$
- กระแสสูงสุด: $I_{max} = V_{max}/Z$
- ถ้าเป็นข้อแรงดันตกคร่อม ต้องดูองค์ประกอบที่ถามให้ชัด

## 4. ของไหล แก๊ส และของแข็ง

### 4.1 ทฤษฎีจลน์ของแก๊ส

- ความเร็ว rms: $v_{rms} = \sqrt{3kT/m}$ หรือ $\sqrt{3RT/M}$
- ระวังมวลต่ออนุภาคกับมวลโมลาร์

### 4.2 แก๊สอุดมคติและอุณหพลศาสตร์

- สมการแก๊สอุดมคติ: $PV = nRT$
- กฎข้อที่ 1: $\Delta U = Q - W$
- กระบวนการต่างๆ มักให้หาค่า $nR$ จากสถานะหนึ่งไปอีกสถานะ

### 4.3 ของไหล

- แรงลอยตัว: $F_b = \rho gV_{displaced}$
- ถ้าวัตถุลอย แรงลอยตัวเท่ากับน้ำหนัก
- ปริมาตรที่จมเกี่ยวข้องโดยตรงกับมวลรวมของวัตถุ

### 4.4 ของแข็ง

- มอดูลัสของยัง: $Y = FL/(A\Delta L)$
- โจทย์ชอบให้เพิ่มมวลหรือแรง แล้วถามการยืดเพิ่ม

## 5. ฟิสิกส์สมัยใหม่

### 5.1 ควอนตัม

- โฟโตอิเล็กทริก: $hf = W + K_{max}$
- เดอบรอยล์: $\lambda = h/p = h/\sqrt{2mK}$
- บทนี้มักเป็นโจทย์ผสม ต้องหา $K$ ก่อนค่อยไปหา $\lambda$

### 5.2 กัมมันตภาพรังสี

- จำนวนที่เหลือ: $N = N_0(1/2)^{t/T}$
- กัมมันตภาพ: $A = \lambda N$
- ถ้าถามตามเวลา ให้ดูว่าต้องใช้ครึ่งชีวิตกี่รอบ

### 5.3 แบบจำลองอะตอมของโบร์

- $E_n = -13.6/n^2$ eV
- สถานะกระตุ้นที่ $k$ หมายถึง $n = k+1$
- พลังงานโฟตอนที่ปล่อยออกมาคิดจากผลต่างระดับพลังงาน

## 6. วิธีอ่านให้จำได้จริง

- อ่านสูตรพร้อมความหมาย ไม่ท่องลอยๆ
- ทำโจทย์ 1 ข้อแล้วต้องสรุปเองให้ได้ว่าใช้สูตรไหนเพราะอะไร
- ถ้าโจทย์แยกเป็น 2 ช่วง ให้แยกวิเคราะห์ก่อนรวมเสมอ
- ทุกครั้งที่ทำโจทย์ยาว ให้เขียน "ตัวแปรที่โจทย์ถาม" ไว้ด้านบน

## 7. แบบฝึกสั้นท้ายบท

1. ถ้าเห็นกราฟ v-t ต้องดูอะไรเป็นอย่างแรก
- พื้นที่ใต้กราฟ

2. ถ้าโจทย์ชนยืดหยุ่นมวลเท่ากัน ทำไมมักใช้เวกเตอร์สามเหลี่ยมได้
- เพราะอนุรักษ์โมเมนตัมและความเร็วหลังชนมักตั้งฉากกัน

3. ถ้าเห็นคำว่า work function และ de Broglie ในข้อเดียว ควรเริ่มจากอะไร
- หา K จากโฟโตอิเล็กทริกก่อน

4. ถ้าระบุท่อปลายปิดเป็น harmonic ที่ 2 ต้องใช้สูตรไหน
- $L = 3\lambda/4$

5. ถ้าตัวเก็บประจุต่อขนานกันแล้วประจุถ่ายเท ใช้หลักอะไร
- อนุรักษ์ประจุ

## 8. สรุปจำสั้น

- กลศาสตร์: ดูกราฟ แรง พลังงาน โมเมนตัม
- คลื่น/เสียง/แสง: จำสมการคลื่น สั่นพ้อง เลนส์ เกรตติง
- ไฟฟ้า/แม่เหล็ก: ดูศักย์ วงจร สนามแม่เหล็ก ฟาราเดย์ คาปาซิเตอร์
- สมัยใหม่: หา K, N, E_n ให้เป็น
