# สรุปเร็วก่อนสอบฟิสิกส์ A-Level 69

เอกสารนี้คือฉบับทวนด่วนก่อนเข้าห้องสอบ ใช้เปิดอ่านรอบสุดท้ายได้เลย

## 1. สูตรที่ต้องจำให้ขึ้นใจ

- การเคลื่อนที่แนวตรง: พื้นที่ใต้กราฟ $v-t$ = การกระจัด
- โปรเจกไทล์: $u_x = u\cos\theta$, $u_y = u\sin\theta$, $t = 2u_y/g$
- แรงเสียดทาน: $f_k = \mu_k N$
- งานและพลังงาน: $W_{net} = \Delta K$, $U_s = \frac{1}{2}kx^2$, $U_g = mgh$
- โมเมนตัม: $p = mv$, $p_{before} = p_{after}$
- SHM: $x = A\sin(\omega t + \phi)$, $v = \omega A\cos(\omega t + \phi)$
- คลื่น: $v = f\lambda$, $T = 1/f$
- เสียง: ท่อปลายปิด $L = (2n-1)\lambda/4$
- dB: $\Delta\beta = 10\log(I_2/I_1)$
- เลนส์: $1/f = 1/s + 1/s'$ 
- ดอปเพลอร์: ระวังเครื่องหมายเข้า/ออก
- ศักย์ไฟฟ้า: $V = kQ/r$, $W = q\Delta V$
- วงจร DC: $V = IR$, $I = \varepsilon/(R+r)$
- สนามแม่เหล็ก: $qvB = mv^2/r$
- ฟาราเดย์: $\varepsilon = |\Delta\Phi/\Delta t|$
- คาปาซิเตอร์: $Q = CV$, $U = Q^2/(2C)$
- แก๊ส: $PV = nRT$, $v_{rms} = \sqrt{3kT/m}$
- ของไหล: $F_b = \rho gV$
- โฟโตอิเล็กทริก: $hf = W + K_{max}$
- de Broglie: $\lambda = h/p = h/\sqrt{2mK}$
- กัมมันตภาพ: $N = N_0(1/2)^{t/T}$
- โบร์: $E_n = -13.6/n^2$ eV

## 2. รูปแบบโจทย์ที่ชอบออก

- กราฟ -> หาพื้นที่และความชัน
- ตัวแปร -> จัดรูปสมการให้ได้
- อัตราส่วน -> ตัดค่าคงที่ออกก่อน
- หน่วย -> แปลงเป็น SI ทุกครั้ง
- สองบทผสม -> หา "ตัวแปรเชื่อม" ให้เจอ

## 3. จุดพลาดที่ต้องระวังมากที่สุด

- ลืมแปลงหน่วย cm, mm, nm, L
- เครื่องหมายผิดในดอปเพลอร์และศักย์ไฟฟ้า
- อ่านคำว่า "เป็นกี่เท่า" แล้วไปหาค่าสัมบูรณ์
- ลืมว่าโจทย์ถามเฉพาะช่วง ไม่ใช่ทั้งระบบ
- ติดข้อยาวเกินเวลา

## 4. ลำดับทำข้อสอบที่แนะนำ

- ข้อที่มองสูตรออกทันที: เก็บก่อน
- ข้อกราฟและบทพื้นฐาน: ทำต่อ
- ข้อจัดรูปยาว: ตั้งสมการไว้ก่อนแล้วค่อยไปต่อ
- ถ้าเกิน 90 วินาทีแล้วยังไม่ไป: ข้าม

## 5. เช็กลิสต์ 5 นาทีสุดท้าย

- จำสูตรแกนแต่ละบทได้หรือไม่
- หน่วย SI พร้อมหรือยัง
- ดอปเพลอร์เข้า/ออกไม่สับสนไหม
- เลนส์และศักย์ไฟฟ้าใส่เครื่องหมายถูกไหม
- ถ้าเป็นอัตราส่วน ตัดค่าคงที่ได้หรือยัง

## 6. สรุปสั้นที่สุด

- กลศาสตร์: ดูกราฟ แรง พลังงาน โมเมนตัม
- คลื่น/เสียง/แสง: จำคลื่น สั่นพ้อง เลนส์ เกรตติง
- ไฟฟ้า/แม่เหล็ก: จำศักย์ วงจร สนามแม่เหล็ก ฟาราเดย์
- สมัยใหม่: จำโฟโตอิเล็กทริก เดอบรอยล์ กัมมันตภาพ โบร์

อ่าน 1 รอบก่อนสอบ = ได้ภาพรวมครบพอสำหรับทวนรอบสุดท้าย
