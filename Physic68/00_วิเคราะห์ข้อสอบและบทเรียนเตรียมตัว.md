## 3) บทเรียนที่ 1 — กลศาสตร์ (Mechanics)

**โจทย์ครอบคลุม:** [physics_motion_forces_problems.md](physics_motion_forces_problems.md), [3-4.md](3-4.md), [5-6.md](5-6.md), [9-10.md](9-10.md) (ข้อ 10), [26-27-28.md](26-27-28.md) (ข้อ 26–27)

### สูตรการเคลื่อนที่แนวตรง (a คงตัว)
| สูตร | ใช้เมื่อ |
|---|---|
| $v = u + at$ | ไม่มี $s$ |
| $s = ut + \tfrac{1}{2}at^2$ | ไม่มี $v$ |
| $v^2 = u^2 + 2as$ | ไม่มี $t$ |
| $s = \dfrac{u+v}{2} t$ | ไม่มี $a$ |

### กฎของนิวตัน
- $\sum \vec F = m\vec a$
- แรงเสียดทาน: $f_s \le \mu_s N$, $f_k = \mu_k N$ (ทิศต้านการเคลื่อนที่)
- **พื้นเอียงทำมุม $\theta$:** $F_{||} = mg\sin\theta$, $N = mg\cos\theta$, $f_k = \mu_k mg\cos\theta$
- ความเร่งลง: $a = g(\sin\theta - \mu_k\cos\theta)$

### งาน-พลังงาน-โมเมนตัม
| ปริมาณ | สูตร |
|---|---|
| งาน | $W = \vec F \cdot \vec s = Fs\cos\theta$ |
| พลังงานจลน์ | $K = \tfrac{1}{2}mv^2$ |
| พลังงานศักย์ | $U = mgh$ (โน้มถ่วง), $\tfrac{1}{2}kx^2$ (สปริง) |
| ทฤษฎีบทงาน-พลังงาน | $W_{\text{net}} = \Delta K$ |
| **งานจากกราฟ $F$–$x$** | = พื้นที่ใต้กราฟ |
| โมเมนตัม | $\vec p = m\vec v$ |
| การดล | $\vec J = \vec F \Delta t = \Delta \vec p$ |
| อนุรักษ์โมเมนตัม (ชน) | $m_1 u_1 + m_2 u_2 = m_1 v_1 + m_2 v_2$ |

**การชน:** ยืดหยุ่นสมบูรณ์ ($\Delta K = 0$) / ไม่ยืดหยุ่น ($K$ ลดลง) / ไม่ยืดหยุ่นเลย (ติดกันหลังชน)

### สมดุลคาน (โมเมนต์ของแรง)
- $\tau = Fd_\perp$ (แรง × แขนของแรง)
- คานสมดุล: $\sum \tau_{\text{cw}} = \sum \tau_{\text{ccw}}$ และ $\sum F = 0$
- **คานสม่ำเสมอ:** น้ำหนักคานตกที่กึ่งกลาง

### การเคลื่อนที่แบบวงกลม
- ความเร่งสู่ศูนย์กลาง: $a_c = \dfrac{v^2}{R} = \omega^2 R$
- แรงสู่ศูนย์กลาง: $F_c = ma_c = \dfrac{mv^2}{R}$
- ความสัมพันธ์: $v = \omega R$, $\omega = 2\pi f = \dfrac{2\pi}{T}$
- **วงกลมระนาบดิ่ง:** ใช้อนุรักษ์พลังงาน $\tfrac{1}{2}mv_{\text{บน}}^2 + mg(2R) = \tfrac{1}{2}mv_{\text{ล่าง}}^2$
- จุดสูงสุด: $F_c = mg + T$ (เชือก) หรือ $F_c = mg$ (เงื่อนไขขั้นต่ำ ไม่ตก)

### SHM (Simple Harmonic Motion)
- มวล-สปริง: $T = 2\pi\sqrt{\dfrac{m}{k}}$
- ลูกตุ้ม: $T = 2\pi\sqrt{\dfrac{L}{g}}$
- $\omega = \sqrt{k/m}$, $v_{\max} = A\omega$, $a_{\max} = A\omega^2$
- พลังงานรวม: $E = \tfrac{1}{2}kA^2$ (คงตัว)

## 4) บทเรียนที่ 2 — คลื่น แสง เสียง

**โจทย์ครอบคลุม:** [7-8.md](7-8.md), [9-10.md](9-10.md) (ข้อ 9), [11-12.md](11-12.md) (ข้อ 11), [26-27-28.md](26-27-28.md) (ข้อ 28)

### สมบัติคลื่นพื้นฐาน
- $v = f\lambda$
- $T = 1/f$
- พลังงานคลื่น $\propto A^2$ (amplitude squared)

### การหักเห (Snell's Law)
$$\dfrac{\sin\theta_1}{\sin\theta_2} = \dfrac{v_1}{v_2} = \dfrac{\lambda_1}{\lambda_2} = \dfrac{n_2}{n_1}$$
**สำคัญ:** มุมที่หน้าคลื่นทำกับรอยต่อ = มุมที่รังสีทำกับเส้นแนวฉาก

### การแทรกสอด (สลิตคู่ของยัง)
- ตำแหน่งสว่าง: $d\sin\theta = n\lambda$
- ตำแหน่งมืด: $d\sin\theta = (n+\tfrac{1}{2})\lambda$
- ความกว้างริ้ว: $W = \dfrac{\lambda L}{d}$
- **เพิ่ม $W$:** เพิ่ม $\lambda$ หรือ $L$ ; ลด $d$

### การเลี้ยวเบน (เกรตติง)
$$d\sin\theta = n\lambda, \quad d = \dfrac{1}{N}$$ ($N$ = เส้นต่อหน่วยความยาว)

### เสียง (ท่อสั่นพ้อง)
| ประเภทท่อ | ความถี่ฮาร์โมนิก | สูตร |
|---|---|---|
| ปลายเปิดทั้งสอง | $n = 1, 2, 3, \ldots$ | $f_n = \dfrac{nv}{2L}$ |
| ปลายปิดข้างหนึ่ง | $n = 1, 3, 5, \ldots$ (เลขคี่) | $f_n = \dfrac{nv}{4L}$ |

**ระยะระหว่างจุดสั่นพ้องต่อเนื่อง = $\lambda/2$**

### เลนส์ / กระจก
$$\dfrac{1}{s} + \dfrac{1}{s'} = \dfrac{1}{f}, \quad m = -\dfrac{s'}{s} = \dfrac{h'}{h}$$
- เลนส์นูน/กระจกเว้า: $f > 0$ (เก็บแสง)
- เลนส์เว้า/กระจกนูน: $f < 0$ (กระจายแสง)
- $s' > 0$: ภาพจริง (กลับหัว) / $s' < 0$: ภาพเสมือน (หัวตั้ง)
- **เลนส์เว้าให้ภาพเสมือนเสมอ** ขนาดเล็กลง อยู่ฝั่งเดียวกับวัตถุ

## 5) บทเรียนที่ 3 — ของไหล ความร้อน อุณหพลศาสตร์

**โจทย์ครอบคลุม:** [17.md](17.md), [18.md](18.md), [19-20.md](19-20.md), [21.md](21.md)

### สภาพยืดหยุ่น (Elasticity)
| ปริมาณ | สูตร | หน่วย |
|---|---|---|
| ความเค้น (Stress) | $\sigma = \dfrac{F}{A}$ | N/m² (Pa) |
| ความเครียด (Strain) | $\varepsilon = \dfrac{\Delta L}{L_0}$ | ไม่มีหน่วย |
| มอดูลัสยัง | $Y = \dfrac{\sigma}{\varepsilon} = \dfrac{FL_0}{A\Delta L}$ | Pa |

### สถิตยศาสตร์ของไหล
- ความดันที่ลึก $h$: $P = P_0 + \rho g h$
- หลักของอาร์คิมีดีส: $F_B = \rho_{\text{ของเหลว}} V_{\text{จม}} g$
- วัตถุลอยนิ่ง: $F_B = mg$
- หลักของปาสคาล: $\dfrac{F_1}{A_1} = \dfrac{F_2}{A_2}$ (ลูกสูบไฮดรอลิก)

### พลศาสตร์ของไหล
- อัตราการไหล (ปริมาตร): $Q = Av$
- สมการความต่อเนื่อง: $A_1 v_1 = A_2 v_2$
- สมการแบร์นูลลี: $P + \tfrac{1}{2}\rho v^2 + \rho g h = \text{คงตัว}$

### ความร้อน
| สูตร | ใช้เมื่อ |
|---|---|
| $Q = mc\Delta T$ | เปลี่ยนอุณหภูมิ |
| $Q = mL$ | เปลี่ยนสถานะ ($L_f$ หลอมเหลว, $L_v$ กลายเป็นไอ) |
| $Q_{\text{สูญเสีย}} = Q_{\text{รับ}}$ | สมดุลความร้อน |

### กฎข้อที่ 1 อุณหพลศาสตร์
$$\Delta U = Q - W$$
- $Q > 0$: ระบบรับความร้อน / $Q < 0$: คายออก
- $W > 0$: ระบบทำงาน (ขยายตัว) / $W < 0$: ถูกทำงาน (หดตัว)
- $W = P\Delta V$ (ความดันคงตัว)
- $\Delta U > 0$: อุณหภูมิเพิ่ม / $\Delta U < 0$: ลด

| กระบวนการ | เงื่อนไข | ผลลัพธ์ |
|---|---|---|
| Isothermal | $T$ คงตัว | $\Delta U = 0$, $Q = W$ |
| Isobaric | $P$ คงตัว | $W = P\Delta V$ |
| Isochoric | $V$ คงตัว | $W = 0$, $\Delta U = Q$ |
| Adiabatic | $Q = 0$ | $\Delta U = -W$ |

## 6) บทเรียนที่ 4 — ไฟฟ้า แม่เหล็ก วงจร

**โจทย์ครอบคลุม:** [11-12.md](11-12.md) (ข้อ 12), [13.md](13.md), [14.md](14.md), [15.md](15.md), [16.md](16.md), [29-30.md](29-30.md) (ข้อ 29)

### ไฟฟ้าสถิต
| สูตร | ความหมาย |
|---|---|
| $F = \dfrac{kq_1 q_2}{r^2}$ | แรงคูลอมบ์ ($k = 9 \times 10^9$ N·m²/C²) |
| $E = \dfrac{kq}{r^2}$ | สนามไฟฟ้าจากจุดประจุ |
| $\vec F = q\vec E$ | แรงบนประจุในสนาม |
| $V = \dfrac{kq}{r}$ | ศักย์ไฟฟ้าจากจุดประจุ |
| $U = \dfrac{kq_1 q_2}{r}$ | พลังงานศักย์ระหว่างประจุคู่ |
| $W_{\text{ภายนอก}} = q(V_B - V_A) = \Delta U$ | งานเลื่อนประจุ |

**คำแนะนำ:** สนามไฟฟ้าเป็น **เวกเตอร์** ต้องวาดทิศก่อน แล้วจึงรวมแบบเวกเตอร์

### ตัวเก็บประจุ
- $Q = CV$
- **อนุกรม:** $\dfrac{1}{C_{eq}} = \dfrac{1}{C_1} + \dfrac{1}{C_2}$ ; $Q$ เท่ากันทุกตัว
- **ขนาน:** $C_{eq} = C_1 + C_2$ ; $V$ เท่ากันทุกตัว
- พลังงานสะสม: $U = \tfrac{1}{2}CV^2 = \tfrac{Q^2}{2C}$

### วงจรไฟฟ้า
- กฎโอห์ม: $V = IR$
- กำลังไฟฟ้า: $P = VI = I^2 R = \dfrac{V^2}{R}$
- **อนุกรม:** $R_{eq} = R_1 + R_2$ ; $I$ เท่ากัน ; $V$ แบ่งตามสัดส่วน $R$
- **ขนาน:** $\dfrac{1}{R_{eq}} = \dfrac{1}{R_1} + \dfrac{1}{R_2}$ ; $V$ เท่ากัน
- กฎเคอร์ชอฟฟ์: $\sum I_{\text{เข้า}} = \sum I_{\text{ออก}}$, $\sum V = 0$ ในวงปิด

### แม่เหล็กไฟฟ้า
| สูตร | ความหมาย |
|---|---|
| $F = qvB\sin\theta$ | แรงบนประจุเคลื่อนที่ |
| $\vec F = I\vec L \times \vec B$ | แรงบนลวดนำกระแส |
| $F = BIL\sin\theta$ | (ขนาด) |
| $\Phi_B = BA\cos\theta$ | ฟลักซ์แม่เหล็ก |
| $\varepsilon = -\dfrac{d\Phi_B}{dt}$ | EMF เหนี่ยวนำ (กฎฟาราเดย์) |
| กฎเลนซ์ | ทิศกระแสเหนี่ยวนำต้านการเปลี่ยนแปลงฟลักซ์ |

**ระวัง:** ใช้กฎมือขวาหาทิศของ $\vec L \times \vec B$ (หรือ $\vec v \times \vec B$)

## 7) บทเรียนที่ 5 — ฟิสิกส์ยุคใหม่ (Modern Physics)

**โจทย์ครอบคลุม:** [22.md](22.md), [23-24.md](23-24.md), [25.md](25.md), [29-30.md](29-30.md) (ข้อ 30)

### โฟตอน
- พลังงานโฟตอน: $E = hf = \dfrac{hc}{\lambda}$
- $h = 6.626 \times 10^{-34}$ J·s
- $hc \approx 1240$ eV·nm (เร็วในการคำนวณ)

### ปรากฏการณ์โฟโตอิเล็กทริก
$$KE_{\max} = hf - W$$
- $W$ = ฟังก์ชันงาน (work function), $W = hf_0$
- ความถี่ขีดเริ่ม $f_0$ : ต่ำกว่านี้ไม่มี photoelectron แม้แสงจะแรง
- **กราฟ $KE_{\max}$–$f$:** ความชัน = $h$, จุดตัดแกนตั้ง = $-W$, จุดตัดแกนนอน = $f_0$

### ทวิภาพคลื่น-อนุภาค
- ความยาวคลื่นเดอบรอยล์: $\lambda = \dfrac{h}{p} = \dfrac{h}{mv}$
- จาก $KE = \tfrac{p^2}{2m}$: $\lambda = \dfrac{h}{\sqrt{2mKE}}$
- ถ้าเร่งด้วย $V$: $KE = qV$ → $\lambda \propto \dfrac{1}{\sqrt{V}}$

### แบบจำลองอะตอมโบร์
| ปริมาณ | สูตร |
|---|---|
| รัศมีวงโคจร | $r_n = n^2 r_1$ ($r_1 = 0.529$ Å) |
| อัตราเร็ว | $v_n = \dfrac{v_1}{n}$ |
| พลังงาน | $E_n = -\dfrac{13.6}{n^2}$ eV |
| โฟตอนที่ปล่อย | $hf = E_i - E_f$ |

**คำเตือน:** "สถานะกระตุ้นที่ 1" คือ $n = 2$ (ไม่ใช่ $n = 1$)

### กัมมันตภาพ (Radioactivity)
- จำนวนนิวเคลียสที่เหลือ: $N(t) = N_0 e^{-\lambda t} = N_0 \left(\tfrac{1}{2}\right)^{t/T_{1/2}}$
- กัมมันตภาพ: $A = \lambda N$ โดย $\lambda = \dfrac{\ln 2}{T_{1/2}}$
- ครึ่งชีวิตผ่านไป $n$ ครั้ง → เหลือ $\dfrac{N_0}{2^n}$
- เลขโมล: $N = n_{\text{mol}} \cdot N_A$ ($N_A = 6.02 \times 10^{23}$)
