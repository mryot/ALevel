# โจทย์ฟังก์ชันกำลังสอง: การหาค่าสูงสุดของรายได้

## โจทย์

สำนักข่าวออนไลน์แห่งหนึ่งมีบริการสำหรับสมาชิกรายเดือน โดยปัจจุบันค่าบริการรายเดือนอยู่ที่ 70 บาทต่อคน และมีสมาชิกอยู่ 18,000 คน นอกจากนี้จากการสำรวจกลุ่มผู้ใช้บริการ พบว่าการปรับราคาค่าบริการจะมีผลต่อจำนวนสมาชิกในรูปแบบฟังก์ชันเชิงเส้น โดยการขึ้นค่าบริการทุก ๆ 1 บาทต่อคน จะทำให้จำนวนสมาชิกลดลง 200 คน (ตัวอย่างเช่น ถ้าขึ้นค่าบริการ 5 บาทต่อคน เป็น 75 บาทต่อคน จะทำให้จำนวนสมาชิกลดลง 1,000 คน เหลือเพียง 17,000 คน)

ให้ $f$ แทนฟังก์ชันรายได้ต่อเดือนของสำนักข่าวเมื่อเก็บค่าบริการ $x$ บาทต่อคน จะได้ว่ากราฟของฟังก์ชัน $f$ มีลักษณะอย่างไร

1. เป็นรูปพาราโบลาหงายที่มีจุดยอดที่ $x = 0$
2. เป็นรูปพาราโบลาหงายที่มีจุดยอดที่ $x = 70$
3. เป็นรูปพาราโบลาคว่ำที่มีจุดยอดที่ $x = 75$
4. เป็นรูปพาราโบลาคว่ำที่มีจุดยอดที่ $x = 80$
5. เป็นกราฟเส้นตรงที่มีค่าความชันเป็นบวก

---

## วิธีทำ

### ขั้นที่ 1: สร้างฟังก์ชันจำนวนสมาชิก

ค่าบริการเพิ่มจาก 70 บาท เป็น $x$ บาท → เพิ่มขึ้น $(x - 70)$ บาท

ทุก 1 บาทที่เพิ่ม → สมาชิกลด 200 คน

$$\text{จำนวนสมาชิก} = 18{,}000 - 200(x - 70)$$

$$= 18{,}000 - 200x + 14{,}000$$

$$= 32{,}000 - 200x$$

**ตรวจสอบ:** เมื่อ $x = 70$ → สมาชิก $= 32{,}000 - 14{,}000 = 18{,}000$ ✓
เมื่อ $x = 75$ → สมาชิก $= 32{,}000 - 15{,}000 = 17{,}000$ ✓

### ขั้นที่ 2: สร้างฟังก์ชันรายได้ $f(x)$

$$f(x) = \text{ค่าบริการ} \times \text{จำนวนสมาชิก}$$

$$f(x) = x(32{,}000 - 200x) = -200x^2 + 32{,}000x$$

### ขั้นที่ 3: วิเคราะห์ลักษณะของกราฟ

$f(x) = ax^2 + bx + c$ โดย $a = -200,\ b = 32{,}000,\ c = 0$

- $a = -200 < 0$ → **พาราโบลาคว่ำ**

### ขั้นที่ 4: หาจุดยอด

สูตรหาจุดยอด: $x = \dfrac{-b}{2a}$

$$x = \dfrac{-32{,}000}{2(-200)} = \dfrac{-32{,}000}{-400} = 80$$

→ จุดยอดอยู่ที่ $x = 80$

### ขั้นที่ 5: หารายได้สูงสุด (เพิ่มเติม)

$$f(80) = -200(80)^2 + 32{,}000(80)$$
$$= -200(6{,}400) + 2{,}560{,}000$$
$$= -1{,}280{,}000 + 2{,}560{,}000 = 1{,}280{,}000 \text{ บาท}$$

ดังนั้นเก็บค่าบริการ 80 บาท/เดือน จะได้รายได้สูงสุด 1,280,000 บาท

---

## คำตอบ

**ข้อ 4.** เป็นรูปพาราโบลาคว่ำที่มีจุดยอดที่ $x = 80$

# เนื้อหาที่ต้องศึกษาเพิ่มเติม

## 1. ฟังก์ชันกำลังสอง (Quadratic Function)

**ฟังก์ชันกำลังสอง** มีรูปทั่วไป:

$$f(x) = ax^2 + bx + c \quad (a \neq 0)$$

โดย $a, b, c$ เป็นค่าคงตัว

กราฟของฟังก์ชันกำลังสองเรียกว่า **พาราโบลา (Parabola)**

---

## 2. ลักษณะของกราฟพาราโบลา

| สัมประสิทธิ์ $a$ | ลักษณะกราฟ | ค่าสุดขีด |
|:---:|:---:|:---:|
| $a > 0$ | พาราโบลา**หงาย** ⌣ | มีจุด**ต่ำสุด** |
| $a < 0$ | พาราโบลา**คว่ำ** ⌢ | มีจุด**สูงสุด** |

---

## 3. จุดยอด (Vertex) ของพาราโบลา

### สูตรหาจุดยอด

ถ้า $f(x) = ax^2 + bx + c$ จุดยอดอยู่ที่:

$$x = \dfrac{-b}{2a}, \quad y = f\!\left(\dfrac{-b}{2a}\right)$$

หรือใช้สูตร $y = c - \dfrac{b^2}{4a}$

### รูปจุดยอด (Vertex Form)

จัดรูปเป็น:
$$f(x) = a(x - h)^2 + k$$

โดย $(h, k)$ คือจุดยอด

- ถ้า $a > 0$: $k$ คือค่าต่ำสุด
- ถ้า $a < 0$: $k$ คือค่าสูงสุด

---

## 4. การหาค่าสูงสุด/ต่ำสุด (Optimization)

### ขั้นตอนทำโจทย์ application

1. **กำหนดตัวแปร** ที่เราต้องการหา (โดยทั่วไปคือสิ่งที่โจทย์ให้ปรับค่า)
2. **สร้างฟังก์ชัน** ที่แทนสิ่งที่ต้องการหาค่าสูงสุด/ต่ำสุด
3. **เขียนเป็นฟังก์ชันกำลังสอง** $f(x) = ax^2 + bx + c$
4. **หาจุดยอด** ด้วยสูตร $x = -\dfrac{b}{2a}$
5. **แทนค่ากลับ** เพื่อหาค่าสูงสุด/ต่ำสุด

### ตัวอย่างทั่วไป: ปัญหาราคา-จำนวน-รายได้

ถ้าราคา $p$ ทำให้จำนวนสินค้าขายได้เป็น $n(p)$ (ลดลงเชิงเส้นเมื่อราคาขึ้น)

$$\text{รายได้} = p \cdot n(p)$$

จะได้ฟังก์ชันกำลังสอง → หาราคาที่ทำให้รายได้สูงสุดด้วยจุดยอด

---

## 5. สมบัติเพิ่มเติมของพาราโบลา

### แกนสมมาตร
แกนสมมาตรของพาราโบลา คือเส้น

$$x = \dfrac{-b}{2a}$$

(เส้นตรงแนวตั้งที่ผ่านจุดยอด)

### จุดตัดแกน $x$ (รากของสมการ)

หาได้จาก $ax^2 + bx + c = 0$ ใช้สูตร:

$$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### Discriminant: $D = b^2 - 4ac$

| ค่า $D$ | จุดตัดแกน $x$ |
|:---:|:---:|
| $D > 0$ | 2 จุด |
| $D = 0$ | 1 จุด (สัมผัสแกน $x$) |
| $D < 0$ | ไม่ตัดแกน $x$ |

### จุดตัดแกน $y$
ที่ $x = 0$ → $y = c$

---

## 6. การจัดรูปเป็น Vertex Form (Completing the Square)

แปลง $f(x) = ax^2 + bx + c$ ให้เป็น $a(x - h)^2 + k$

### ตัวอย่าง
$$f(x) = 2x^2 - 8x + 5$$

$$= 2(x^2 - 4x) + 5$$

$$= 2(x^2 - 4x + 4 - 4) + 5$$

$$= 2(x - 2)^2 - 8 + 5$$

$$= 2(x - 2)^2 - 3$$

จุดยอด $(2, -3)$ และเป็นค่าต่ำสุด (เพราะ $a = 2 > 0$)

---

## 7. โจทย์ฝึกเพิ่มเติม พร้อมเฉลย

### ข้อ 1
ร้านอาหารแห่งหนึ่งขายข้าวกล่อง ราคาปัจจุบัน 50 บาท ขายได้ 200 กล่อง/วัน ถ้าลดราคาทุก 1 บาท จะขายได้เพิ่ม 10 กล่อง ควรขายราคาเท่าใดจึงได้รายได้สูงสุด

**เฉลย:**
ให้ $x$ = ราคา → จำนวนที่ขายได้ $= 200 + 10(50 - x) = 700 - 10x$
$f(x) = x(700 - 10x) = -10x^2 + 700x$
จุดยอด: $x = \dfrac{-700}{-20} = 35$ บาท
รายได้สูงสุด $= 35 \times 350 = 12{,}250$ บาท

### ข้อ 2
หาจุดยอดและค่าสูงสุด/ต่ำสุดของ $f(x) = -3x^2 + 12x - 7$

**เฉลย:**
$x = \dfrac{-12}{-6} = 2$, $f(2) = -12 + 24 - 7 = 5$
จุดยอด $(2, 5)$ เป็นค่า**สูงสุด** (เพราะ $a = -3 < 0$)

### ข้อ 3
จงหาว่ากราฟของ $f(x) = x^2 - 6x + 10$ ตัดแกน $x$ หรือไม่

**เฉลย:**
$D = 36 - 40 = -4 < 0$ → **ไม่ตัดแกน $x$**

### ข้อ 4
กำแพงยาว 60 เมตร ต้องการกั้นรั้วเป็นรูปสี่เหลี่ยมผืนผ้าโดยใช้กำแพงเป็นด้านหนึ่ง จงหาพื้นที่สูงสุดที่กั้นได้

**เฉลย:**
ให้ด้านที่ขนานกำแพงยาว $x$ ด้านที่ตั้งฉากยาว $y$
$x + 2y = 60 \Rightarrow x = 60 - 2y$
พื้นที่ $A = xy = (60 - 2y)y = -2y^2 + 60y$
จุดยอด: $y = \dfrac{-60}{-4} = 15$
$A_{\max} = -2(225) + 900 = 450$ ตร.ม.
