# โจทย์เอ็กซ์โพเนนเชียลและลอการิทึม

## โจทย์

กำหนดให้ $x$ เป็นจำนวนจริงบวกที่สอดคล้องกับสมการ

$$e^{x^e} = e^e$$

ค่าของ $\ln(\ln x)$ ตรงกับข้อใดต่อไปนี้

1. $0$
2. $1$
3. $-1$
4. $e$
5. $-e$

---

## วิธีทำ

### ขั้นที่ 1: แก้สมการหา $x$

$$e^{x^e} = e^e$$

ฐานทั้งสองข้างเป็น $e$ เหมือนกัน ดังนั้นจับเลขชี้กำลังมาเท่ากัน:

$$x^e = e$$

ยกกำลัง $\dfrac{1}{e}$ ทั้งสองข้าง:

$$\left(x^e\right)^{1/e} = e^{1/e}$$

$$x = e^{1/e}$$

### ขั้นที่ 2: หา $\ln x$

$$\ln x = \ln\!\left(e^{1/e}\right) = \frac{1}{e} \cdot \ln e = \frac{1}{e} = e^{-1}$$

### ขั้นที่ 3: หา $\ln(\ln x)$

$$\ln(\ln x) = \ln\!\left(e^{-1}\right) = -1 \cdot \ln e = -1$$

---

## คำตอบ

**ข้อ 3. $-1$**

---

# เนื้อหาที่ควรทบทวนเพิ่มเติม

## 1. สมบัติเลขยกกำลัง

สำหรับ $a > 0$:

$$\left(a^m\right)^n = a^{mn}, \quad a^m \cdot a^n = a^{m+n}, \quad \frac{a^m}{a^n} = a^{m-n}$$

## 2. การแก้สมการเอ็กซ์โพเนนเชียลที่ฐานเท่ากัน

ถ้า $a > 0, a \neq 1$ และ $a^M = a^N$ แล้ว $M = N$

นั่นคือเมื่อฐานเท่ากัน เลขชี้กำลังต้องเท่ากันด้วย

## 3. สมบัติของลอการิทึมธรรมชาติ ($\ln$)

$$\ln e = 1, \quad \ln 1 = 0$$
$$\ln(a^k) = k\ln a$$
$$\ln(ab) = \ln a + \ln b, \quad \ln\!\left(\tfrac{a}{b}\right) = \ln a - \ln b$$
$$e^{\ln x} = x \quad (x > 0), \quad \ln(e^x) = x$$

## 4. โดเมนของ $\ln(\ln x)$

- $\ln x$ นิยามเมื่อ $x > 0$
- $\ln(\ln x)$ นิยามเมื่อ $\ln x > 0$ คือ $x > 1$

ในโจทย์นี้ $x = e^{1/e} \approx 1.4447 > 1$ ✓ (จึงคำนวณได้)

แต่สังเกตว่า $\ln x = \tfrac{1}{e} < 1$ ดังนั้น $\ln(\ln x) < 0$ ตามที่คำนวณได้ $= -1$

## 5. ข้อควรระวังในการอ่านโจทย์

อย่าสับสนระหว่าง:
- $x^e = e^e$ → $x = e$ → $\ln(\ln x) = 0$
- $e^{x^e} = e^e$ → $x^e = e$ → $x = e^{1/e}$ → $\ln(\ln x) = -1$

การอ่านสัญลักษณ์ผิดเพี้ยนจะทำให้ได้คำตอบต่างกันโดยสิ้นเชิง
