# โจทย์โดเมนของฟังก์ชันประกอบ

## โจทย์

ให้ $f$ และ $g$ เป็นฟังก์ชันที่นิยามโดย

$$f(x) = \sqrt{2 - x} \qquad \text{และ} \qquad g(x) = \log_3(5 - x)$$

จะได้ว่า โดเมนของฟังก์ชัน $f \circ g$ ตรงกับข้อใดต่อไปนี้

1. $(-\infty, 5)$
2. $(-\infty, 2]$
3. $(-3, 2]$
4. $[-4, 5)$
5. $[-4, \infty)$

---

## หลักการสำคัญ

โดเมนของ $f \circ g$ คือ

$$D_{f \circ g} = \{\,x \mid x \in D_g \text{ และ } g(x) \in D_f\,\}$$

นั่นคือต้องสอดคล้องกับ **เงื่อนไขสองอย่างพร้อมกัน**:
1. $x$ อยู่ในโดเมนของ $g$
2. ค่า $g(x)$ ต้องอยู่ในโดเมนของ $f$

---

## วิธีทำ

### ขั้นที่ 1: หาโดเมนของ $g$

$g(x) = \log_3(5 - x)$ นิยามเมื่ออาร์กิวเมนต์ของ $\log$ เป็นบวก:

$$5 - x > 0 \implies x < 5 \qquad \cdots (1)$$

ดังนั้น $D_g = (-\infty, 5)$

### ขั้นที่ 2: หาโดเมนของ $f$

$f(t) = \sqrt{2 - t}$ นิยามเมื่อค่าใต้รากไม่ติดลบ:

$$2 - t \geq 0 \implies t \leq 2$$

ดังนั้น $D_f = (-\infty, 2]$

### ขั้นที่ 3: บังคับให้ $g(x) \in D_f$

ต้องการ $g(x) \leq 2$:

$$\log_3(5 - x) \leq 2$$

เนื่องจากฐาน $3 > 1$ ฟังก์ชัน $\log_3$ เป็นฟังก์ชันเพิ่ม (เครื่องหมายอสมการคงเดิม):

$$5 - x \leq 3^2$$

$$5 - x \leq 9$$

$$-x \leq 4$$

$$x \geq -4 \qquad \cdots (2)$$

### ขั้นที่ 4: รวม (1) และ (2)

$$x < 5 \quad \text{และ} \quad x \geq -4$$

$$\therefore -4 \leq x < 5$$

---

## คำตอบ

**ข้อ 4.** $[-4, 5)$

---

## ตรวจสอบที่จุดสำคัญ

**ที่ $x = -4$:**
- $g(-4) = \log_3(9) = 2$ ✓ (อยู่ใน $D_f$)
- $f(g(-4)) = \sqrt{2 - 2} = 0$ ✓

**ที่ $x \to 5^{-}$:**
- $5 - x \to 0^{+}$ ทำให้ $\log_3(5 - x) \to -\infty$
- $f(g(x)) = \sqrt{2 - g(x)} \to \infty$ (นิยามได้แต่ค่าโตขึ้นเรื่อย ๆ)

**ที่ $x = 5$:**
- $\log_3(0)$ ไม่นิยาม → ไม่อยู่ในโดเมน

**ที่ $x = -5$:**
- $g(-5) = \log_3(10) > 2$ → ไม่อยู่ใน $D_f$ → ไม่อยู่ในโดเมน

# เนื้อหาที่ควรทบทวนเพิ่มเติม

## 1. ฟังก์ชันประกอบ (Composite Function)

$$(f \circ g)(x) = f(g(x))$$

อ่าน "$f$ ประกอบ $g$" หรือ "$f$ หลังจาก $g$" — คือนำ $x$ ใส่ใน $g$ ก่อน แล้วเอาผลลัพธ์ไปใส่ใน $f$

### โดเมนของฟังก์ชันประกอบ

$$D_{f \circ g} = \{\,x \in D_g \mid g(x) \in D_f\,\}$$

**สูตรลัด:** หา 2 เงื่อนไขแล้วรวม intersection
1. $x$ ต้องอยู่ในโดเมนของ $g$
2. $g(x)$ ต้องอยู่ในโดเมนของ $f$

---

## 2. โดเมนของฟังก์ชันมาตรฐาน

| ฟังก์ชัน | เงื่อนไขโดเมน |
|---|---|
| $\sqrt{f(x)}$ | $f(x) \geq 0$ |
| $\sqrt[2n]{f(x)}$ | $f(x) \geq 0$ |
| $\sqrt[2n+1]{f(x)}$ | จำนวนจริงทุกค่า |
| $\dfrac{1}{f(x)}$ | $f(x) \neq 0$ |
| $\log_a f(x)$ | $f(x) > 0,\ a > 0,\ a \neq 1$ |
| $a^{f(x)}$ | จำนวนจริงทุกค่า ($a > 0$) |
| $\tan x$ | $x \neq \dfrac{\pi}{2} + k\pi$ |

---

## 3. สมบัติของลอการิทึมที่ใช้แก้อสมการ

### กรณีฐาน $a > 1$ (เช่น $\log_2, \log_3, \log_{10}, \ln$)

$\log_a$ เป็น**ฟังก์ชันเพิ่ม** เครื่องหมายอสมการ**คงเดิม**:
$$\log_a M < \log_a N \iff M < N$$

### กรณีฐาน $0 < a < 1$ (เช่น $\log_{1/2}$)

$\log_a$ เป็น**ฟังก์ชันลด** เครื่องหมายอสมการ**กลับด้าน**:
$$\log_a M < \log_a N \iff M > N$$

### การแปลง $\log_a x \leq k$ เป็นไม่มี $\log$

ถ้า $a > 1$:
$$\log_a x \leq k \iff 0 < x \leq a^k$$

(ต้องไม่ลืมเงื่อนไข $x > 0$)

---

## 4. การแก้อสมการที่มี $\log$

### ขั้นตอน
1. **หาเงื่อนไขโดเมน** ของ $\log$ ทุกตัว ($> 0$)
2. **แปลงเป็นรูปไม่มี $\log$** โดยใช้ฐานกับเครื่องหมายให้ถูก
3. **แก้อสมการ** ที่ได้
4. **รวม intersection** ของเงื่อนไขทั้งหมด

### ตัวอย่าง: แก้ $\log_3(5 - x) \leq 2$

1. โดเมน: $5 - x > 0 \Rightarrow x < 5$
2. แปลง: $5 - x \leq 3^2 = 9 \Rightarrow x \geq -4$
3. รวม: $-4 \leq x < 5$

---

## 5. โจทย์ฝึกเพิ่มเติม พร้อมเฉลย

### ข้อ 1
ให้ $f(x) = \sqrt{x - 1}$ และ $g(x) = \dfrac{1}{x - 2}$ จงหาโดเมนของ $f \circ g$

**เฉลย:**
- $D_g$: $x \neq 2$
- $g(x) \in D_f$: $\dfrac{1}{x-2} \geq 1 \iff \dfrac{1}{x-2} - 1 \geq 0 \iff \dfrac{3-x}{x-2} \geq 0$
  - แก้ได้: $2 < x \leq 3$
- รวม: $D_{f \circ g} = (2, 3]$

### ข้อ 2
ให้ $f(x) = \log_2 x$ และ $g(x) = 4 - x^2$ จงหาโดเมนของ $f \circ g$

**เฉลย:**
- $D_g$: ทุก $x \in \mathbb{R}$
- $g(x) > 0$: $4 - x^2 > 0 \iff -2 < x < 2$
- รวม: $D_{f \circ g} = (-2, 2)$

### ข้อ 3
ให้ $f(x) = \sqrt{4 - x}$ และ $g(x) = x^2$ จงหาโดเมนของ $f \circ g$

**เฉลย:**
- $D_g$: ทุก $x \in \mathbb{R}$
- $g(x) \leq 4$: $x^2 \leq 4 \iff -2 \leq x \leq 2$
- รวม: $D_{f \circ g} = [-2, 2]$
