# โจทย์ลำดับเวียนเกิด

## โจทย์

ให้ $a_n$ เป็นลำดับที่สอดคล้องกับ

$$a_{n+1} = a_n + \frac{n}{2} \quad \text{สำหรับทุกจำนวนบวก } n$$

ถ้า $a_{30} = 219$ แล้ว $a_1$ มีค่าเท่าใด

ตัวเลือก:

1. $-29$
2. $-\dfrac{27}{2}$
3. $\dfrac{3}{2}$
4. $16$
5. $30$

---

## วิธีทำ

จากสมการเวียนเกิด

$$a_{n+1} - a_n = \frac{n}{2}$$

ให้บวกตั้งแต่ $n=1$ ถึง $n=29$ จะได้

$$\sum_{n=1}^{29}(a_{n+1}-a_n)=\sum_{n=1}^{29}\frac{n}{2}$$

ด้านซ้ายเป็นผลบวกแบบยุบพจน์ (telescoping):

$$a_{30}-a_1 = \frac{1}{2}\sum_{n=1}^{29} n$$

ใช้สูตรผลบวกจำนวนเต็มบวก

$$\sum_{n=1}^{29} n = \frac{29\cdot 30}{2} = 435$$

ดังนั้น

$$a_{30}-a_1 = \frac{435}{2}$$

แทน $a_{30}=219$:

$$219-a_1 = \frac{435}{2}$$

$$a_1 = 219 - \frac{435}{2} = \frac{438-435}{2} = \frac{3}{2}$$

---

## คำตอบ

**ข้อ 3.** $\dfrac{3}{2}$

---

## ตรวจสอบย้อนกลับอย่างเร็ว

ถ้า $a_1=\dfrac{3}{2}$ แล้ว

$$a_{30}=a_1+\sum_{n=1}^{29}\frac{n}{2}=\frac{3}{2}+\frac{435}{2}=\frac{438}{2}=219$$

ตรงกับโจทย์ ✓
