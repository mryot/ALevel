## 3) สูตรหลักต้องท่อง (Cheat Sheet)

### 3.1 เซต / ตรรกศาสตร์
- $n(A\cup B) = n(A) + n(B) - n(A\cap B)$
- $n(A\cup B\cup C) = \sum n - \sum n(\cap_{2}) + n(A\cap B\cap C)$
- De Morgan: $(A\cup B)^c = A^c\cap B^c$, $(A\cap B)^c = A^c\cup B^c$
- เพาเวอร์เซต: $|P(A)| = 2^{n(A)}$
- ค่าความจริง: **Tautology** = T ทุกกรณี, **Contradiction** = F ทุกกรณี, **Contingency** = มีทั้ง T และ F

### 3.2 พีชคณิต / ฟังก์ชัน
- พาราโบลา $y=ax^2+bx+c$ จุดยอด $x=-\dfrac{b}{2a}$
- $|f(x)|\le k \iff -k\le f(x)\le k$ (ต้อง $k\ge 0$)
- โดเมนของ $g\circ f$: ต้อง $x\in D_f$ **และ** $f(x)\in D_g$
- ทฤษฎีบทเศษ: $P(x)$ หารด้วย $x-c$ เหลือเศษ $P(c)$
- $\log_a b = \dfrac{\ln b}{\ln a}$, $a^{\log_a x}=x$, $\log(ab)=\log a+\log b$
- $|z|^2 = z\bar{z}$, $|z_1 z_2|=|z_1||z_2|$, $\left|\dfrac{z_1}{z_2}\right|=\dfrac{|z_1|}{|z_2|}$

### 3.3 เรขาคณิต / เวกเตอร์
- วงรี $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1$ จุดตัดแกน Y: $(0,\pm b)$
- ระยะห่างจุดถึงเส้น $ax+by+c=0$: $d=\dfrac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}$
- พื้นที่สามเหลี่ยมจากเวกเตอร์: $\text{Area}=\tfrac12\|\vec{u}\times\vec{v}\|$
- $\vec{u}\cdot\vec{v}=\|\vec u\|\|\vec v\|\cos\theta$
- เวกเตอร์หนึ่งหน่วย: $\hat{u}=\dfrac{\vec u}{\|\vec u\|}$

### 3.4 ตรีโกณมิติ
- กฎ sine: $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=2R$
- กฎ cosine: $a^2=b^2+c^2-2bc\cos A$
- $\sin^2\theta+\cos^2\theta=1$, $1+\tan^2\theta=\sec^2\theta$
- $\cot\theta-\tan\theta = \dfrac{\cos^2\theta-\sin^2\theta}{\sin\theta\cos\theta}=\dfrac{2\cos 2\theta}{\sin 2\theta}=2\cot 2\theta$

### 3.5 แคลคูลัส / ลิมิต
- $\displaystyle\lim_{x\to a}f(x)$ มีอยู่ ⟺ ลิมิตซ้าย = ลิมิตขวา
- กฎผลรวม/ผลคูณของลิมิต ใช้ได้เมื่อ **ทั้งสองลิมิตหาค่าได้**
- $(fg)'=f'g+fg'$, $\left(\dfrac{f}{g}\right)'=\dfrac{f'g-fg'}{g^2}$
- Chain rule: $(f(g(x)))' = f'(g(x))\cdot g'(x)$
- FTC: $\displaystyle\int_a^b f(x)\,dx = F(b)-F(a)$
- การเปลี่ยนตัวแปร: $\int f(g(x))g'(x)\,dx = \int f(u)\,du$

### 3.6 ลำดับ-อนุกรม / ดอกเบี้ย
- เลขคณิต: $a_n=a_1+(n-1)d$, $S_n=\dfrac{n}{2}(a_1+a_n)$
- เรขาคณิต: $a_n=a_1 r^{n-1}$, $S_n=\dfrac{a_1(1-r^n)}{1-r}$
- อนุกรมเรขาคณิตอนันต์ ($|r|<1$): $S_\infty=\dfrac{a_1}{1-r}$
- ดอกเบี้ยทบต้น: $A=P(1+i)^n$ โดย $i=\dfrac{r}{k}$, $n=k\cdot t$

### 3.7 เมทริกซ์
- การดำเนินการตามแถว: สลับแถว เปลี่ยนเครื่องหมาย det, คูณแถวด้วย $k$ → det คูณ $k$, บวกแถวอื่นเข้ามา → det คงเดิม
- $\det(AB)=\det A\det B$, $\det(A^T)=\det A$, $\det(kA_{n\times n})=k^n\det A$
- $A$ มีอินเวอร์ส ⟺ $\det A\neq 0$

### 3.8 ความน่าจะเป็น / การนับ
- กฎคูณ: ทำต่อกัน → คูณ; กฎบวก: เป็นทางเลือกที่ **ไม่เกิดร่วมกัน** → บวก
- จัดเรียง: $P(n,r)=\dfrac{n!}{(n-r)!}$, จัดหมู่: $C(n,r)=\dfrac{n!}{r!(n-r)!}$
- Binomial: $P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}$
- ส่วนเติมเต็ม: $P(A) = 1-P(A^c)$ (ใช้กับ "อย่างน้อย 1")
- $P(A\cup B)=P(A)+P(B)-P(A\cap B)$

### 3.9 สถิติ / การแจกแจงปกติ
- ค่าเฉลี่ย: $\bar{x}=\dfrac{\sum x_i}{n}$ → $\sum x_i = n\bar{x}$
- การแจกแจงปกติ: คำนวณ $z=\dfrac{x-\mu}{\sigma}$ แล้วเปิดตาราง
- กฎ 68-95-99.7 (1σ, 2σ, 3σ)

## 4) กับดักที่พบบ่อยในข้อสอบ (Common Traps)

| ลำดับ | กับดัก | วิธีระวัง |
|---|---|---|
| 1 | โดเมนของฟังก์ชันประกอบ ลืมเงื่อนไข $f(x)\in D_g$ | เช็กทั้ง **สองชั้น** เสมอ |
| 2 | $|f(x)|=k$ มี $\pm$ ลืมแยกกรณี | แยกสองสมการ + ตรวจคำตอบ |
| 3 | ลิมิตของผลคูณ ทั้งสองตัวต้องมีลิมิต | ถ้าตัวใดไม่มี อาจมีคำตอบ "หาค่าไม่ได้" |
| 4 | เวกเตอร์ 3 มิติ ลืมใช้ cross product หาพื้นที่ | สูตร $\tfrac12\|\vec u\times\vec v\|$ |
| 5 | ลำดับเรขาคณิตอนันต์ ลืมเช็ก $|r|<1$ | ไม่งั้นไม่ลู่เข้า |
| 6 | ดีเทอร์มิแนนต์ สลับแถว = เปลี่ยนเครื่องหมาย | จำกฎ row op ทั้ง 3 ข้อ |
| 7 | "อย่างน้อย 1" คิดตรง ๆ ยาก | ใช้ $1-P(\text{ไม่มีเลย})$ |
| 8 | การแจกแจงปกติ ลืมเปิด z = $(x-\mu)/\sigma$ ก่อน | Standardize ทุกครั้ง |
| 9 | ตรรกศาสตร์: Contingency ≠ Contradiction | T บ้าง F บ้าง ไม่ใช่ F หมด |
| 10 | จำนวนเชิงซ้อน: $|z|^2 = z\bar z$ ไม่ใช่ $z^2$ | คูณกับคอนจูเกตเสมอ |
