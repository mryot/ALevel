# โจทย์ทฤษฎีเซต: ส่วนเติมเต็มและแผนภาพเวนน์

## โจทย์

กำหนดให้ $A, B, C$ เป็นเซต โดยที่

$$n(A) = 40, \quad n(A \cap B) = 27, \quad n(A \cap C) = 21, \quad n(A \cap B \cap C) = 10$$

จงหาจำนวนสมาชิกของเซต $A - (B \cup C)$

---

## วิธีทำอย่างละเอียด

### ขั้นตอนที่ 1: ทำความเข้าใจเซต $A - (B \cup C)$

เซต $A - (B \cup C)$ หมายถึง **สมาชิกที่อยู่ใน $A$ แต่ไม่อยู่ใน $B$ และไม่อยู่ใน $C$**

พูดอีกอย่างคือ:
$$A - (B \cup C) = \{x \in A : x \notin B \text{ และ } x \notin C\}$$

หรือเขียนได้ว่า:
$$n(A - (B \cup C)) = n(A) - n(A \cap (B \cup C))$$

---

### ขั้นตอนที่ 2: หาจำนวนสมาชิกของ $A \cap (B \cup C)$

$A \cap (B \cup C)$ หมายถึง สมาชิกที่อยู่ใน $A$ และอยู่ในอย่างน้อยหนึ่งเซตของ $B$ หรือ $C$

นั่นคือ: $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

ใช้หลักการรวม-แยก (Inclusion-Exclusion Principle):
$$n((A \cap B) \cup (A \cap C)) = n(A \cap B) + n(A \cap C) - n(A \cap B \cap C)$$

แทนค่า:
$$n(A \cap (B \cup C)) = 27 + 21 - 10 = 38$$

---

### ขั้นตอนที่ 3: คำนวณ $n(A - (B \cup C))$

$$n(A - (B \cup C)) = n(A) - n(A \cap (B \cup C))$$

$$= 40 - 38 = 2$$

---

### ขั้นตอนที่ 4: ตรวจสอบด้วยการแบ่งเซต $A$

ลองแบ่งเซต $A$ ออกเป็นส่วนต่างๆ ตามแผนภาพเวนน์:

1. **สมาชิกใน $A \cap B$ แต่ไม่ใน $C$**:
   $$n(A \cap B \cap C^c) = n(A \cap B) - n(A \cap B \cap C) = 27 - 10 = 17$$

2. **สมาชิกใน $A \cap C$ แต่ไม่ใน $B$**:
   $$n(A \cap C \cap B^c) = n(A \cap C) - n(A \cap B \cap C) = 21 - 10 = 11$$

3. **สมาชิกใน $A \cap B \cap C$**:
   $$n(A \cap B \cap C) = 10$$

4. **สมาชิกใน $A$ แต่ไม่ใน $B$ หรือ $C$**:
   $$n(A \cap B^c \cap C^c) = n(A) - [n(A \cap B \cap C^c) + n(A \cap C \cap B^c) + n(A \cap B \cap C)]$$
   $$= 40 - (17 + 11 + 10) = 40 - 38 = 2$$

**ตรวจสอบ**: $17 + 11 + 10 + 2 = 40$ ✓

# เนื้อหาเพื่อศึกษาเพิ่มเติม

## 1) ส่วนเติมเต็มของเซต (Set Complement)

ส่วนเติมเต็มของเซต $B$ เทียบกับเซต $A$ คือ:
$$A - B = \{x \in A : x \notin B\}$$

สมบัติ:
- $n(A - B) = n(A) - n(A \cap B)$
- $A - B = A \cap B^c$ (โดยที่ $B^c$ คือส่วนเติมเต็มของ $B$)

**ตัวอย่าง**: ถ้า $A = \{1, 2, 3, 4, 5\}$ และ $B = \{3, 4, 5, 6\}$

แล้ว $A - B = \{1, 2\}$ ซึ่ง $n(A - B) = 2$

---

## 2) ยูเนียนและอินเตอร์เซกชัน

### ยูเนียน (Union)
$$A \cup B = \{x : x \in A \text{ หรือ } x \in B \text{ หรือ ทั้งคู่}\}$$

### อินเตอร์เซกชัน (Intersection)
$$A \cap B = \{x : x \in A \text{ และ } x \in B\}$$

### ความสัมพันธ์
$$B \cup C = \{x : x \in B \text{ หรือ } x \in C\}$$

$$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

---

## 3) หลักการรวม-แยก (Inclusion-Exclusion Principle)

สำหรับสองเซต:
$$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$

สำหรับสามเซต:
$$n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(A \cap C) - n(B \cap C) + n(A \cap B \cap C)$$

**ค่อย: เพิ่มกรณีเดี่ยว → ลบกรณีคู่ → เพิ่มกรณีสาม**

---

## 4) การแบ่งเซต $A$ ตามแผนภาพเวนน์ (สำหรับสามเซต)

เซต $A$ สามารถแบ่งออกเป็น 8 ส่วน (ภูมิภาค) ต่างกัน:

1. $A \cap B \cap C$ (อยู่ในทั้งสามเซต)
2. $A \cap B \cap C^c$ (อยู่ใน $A$ และ $B$ แต่ไม่ใน $C$)
3. $A \cap B^c \cap C$ (อยู่ใน $A$ และ $C$ แต่ไม่ใน $B$)
4. $A^c \cap B \cap C$ (อยู่ใน $B$ และ $C$ แต่ไม่ใน $A$)
5. $A \cap B^c \cap C^c$ (อยู่ใน $A$ เท่านั้น)
6. $A^c \cap B \cap C^c$ (อยู่ใน $B$ เท่านั้น)
7. $A^c \cap B^c \cap C$ (อยู่ใน $C$ เท่านั้น)
8. $A^c \cap B^c \cap C^c$ (ไม่อยู่ในเซตใด)

---

## 5) สูตรอินเตอร์เซกชันและยูเนียน

สำหรับสองเซต (กรณี $(A \cap B) \cup (A \cap C)$):
$$n((A \cap B) \cup (A \cap C)) = n(A \cap B) + n(A \cap C) - n((A \cap B) \cap (A \cap C))$$

$$= n(A \cap B) + n(A \cap C) - n(A \cap B \cap C)$$

การจำง่ายๆ: **เพิ่ม + เพิ่ม - ลบส่วนซ้ำ**

---

## 6) ความเห็นทั่วไปเกี่ยวกับ $A - (B \cup C)$

มีวิธีคิดหลายแบบ:

**วิธีที่ 1 (ใช้ส่วนเติมเต็ม)**:
$$A - (B \cup C) = A \cap (B \cup C)^c$$

โดยใช้กฎของเดอมอร์แกน (De Morgan's Law):
$$(B \cup C)^c = B^c \cap C^c$$

ดังนั้น:
$$A - (B \cup C) = A \cap B^c \cap C^c$$

**วิธีที่ 2 (ใช้การลบ)**:
$$n(A - (B \cup C)) = n(A) - n(A \cap (B \cup C))$$

$$= n(A) - n((A \cap B) \cup (A \cap C))$$

$$= n(A) - [n(A \cap B) + n(A \cap C) - n(A \cap B \cap C)]$$

---

## 7) กลยุทธ์แก้โจทย์ประเภทนี้

### กลยุทธ์หลัก

**ขั้นตอน 1: ระบุสิ่งที่ต้องหา**
- เข้าใจความหมายของนิพจน์เซต เช่น $A - (B \cup C)$ หมายถึงอะไร

**ขั้นตอน 2: แยกเป็นส่วนที่ง่ายกว่า**
- $A - (B \cup C) = A \cap (B \cup C)^c = A \cap B^c \cap C^c$
- หรือ: $n(A - (B \cup C)) = n(A) - n(A \cap (B \cup C))$

**ขั้นตอน 3: ใช้หลักการรวม-แยก**
- $n(A \cap (B \cup C)) = n(A \cap B) + n(A \cap C) - n(A \cap B \cap C)$

**ขั้นตอน 4: ตรวจสอบ**
- วาดแผนภาพเวนน์เพื่อตรวจสอบว่าส่วนต่างๆ รวมกันได้เท่ากับ $n(A)$ หรือไม่

### เคล็ดลับเพิ่มเติม

1. **ใช้กฎของเดอมอร์แกน**: $(B \cup C)^c = B^c \cap C^c$
2. **แบ่งเซต**: ลองแบ่ง $A$ ออกเป็นส่วนต่างๆ ตามแผนภาพเวนน์
3. **ตรวจสอบด้วยตัวอักษร**: เริ่มด้วยตัวอักษร/ตัวเลข เพื่อตรวจสอบคำตอบ

---

## 8) โจทย์ฝึกเพิ่มเติม

### ข้อฝึก 1
ให้ $A, B, C$ เป็นเซต ที่ $n(A) = 50$, $n(A \cap B) = 20$, $n(A \cap C) = 15$, $n(A \cap B \cap C) = 8$

จงหา $n(A - (B \cup C))$

เฉลยย่อ:
- $n(A \cap (B \cup C)) = 20 + 15 - 8 = 27$
- $n(A - (B \cup C)) = 50 - 27 = 23$

### ข้อฝึก 2
เซต $A$ มี 60 สมาชิก เซต $B$ มี 40 สมาชิก และ $A \cap B$ มี 25 สมาชิก

จงหา $n(A - B)$

เฉลยย่อ:
- $n(A - B) = n(A) - n(A \cap B) = 60 - 25 = 35$

### ข้อฝึก 3
ให้ $n(B \cup C) = 35$, $n(B) = 20$, $n(C) = 18$

จงหา $n(B \cap C)$

เฉลยย่อ:
- ใช้หลักการรวม-แยก: $n(B \cup C) = n(B) + n(C) - n(B \cap C)$
- $35 = 20 + 18 - n(B \cap C)$
- $n(B \cap C) = 3$

### ข้อฝึก 4
กำหนด $n(A) = 100$, $n(B) = 60$, $n(C) = 50$, $n(A \cap B) = 30$, $n(A \cap C) = 25$, $n(B \cap C) = 20$, $n(A \cap B \cap C) = 10$

จงหา $n((A - B) - C)$

เฉลยย่อ:
- $(A - B) - C = A \cap B^c \cap C^c$
- $n(A - B) = n(A) - n(A \cap B) = 100 - 30 = 70$
- $n((A - B) \cap C) = n(A \cap C) - n(A \cap B \cap C) = 25 - 10 = 15$
- $n((A - B) - C) = n(A - B) - n((A - B) \cap C) = 70 - 15 = 55$
