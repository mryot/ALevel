# โจทย์จำนวนเชิงซ้อน: ค่าสัมบูรณ์ของนิพจน์

## โจทย์

ถ้า

$$z_1 = 3 - 2i \qquad \text{และ} \qquad z_2 = 1 + 5i$$

แล้ว

$$|z_1^2 + 2z_1z_2 + z_2^2|$$

มีค่าเท่าใด

---

## วิธีทำอย่างละเอียด (เคล็ดลับเด็ด!)

### ขั้นตอนที่ 1: จัดรูปพหุนามข้างในค่าสัมบูรณ์

สังเกตว่าพจน์ $z_1^2 + 2z_1z_2 + z_2^2$ มันเข้าสูตร **กำลังสองสมบูรณ์** เป๊ะๆ เลย คือ:

$$(หน้า)^2 + 2(หน้า)(หลัง) + (หลัง)^2 = (หน้า + หลัง)^2$$

ดังนั้น เราสามารถรวบพจน์ได้แบบนี้:

$$|z_1^2 + 2z_1z_2 + z_2^2| = |(z_1 + z_2)^2|$$

---

### ขั้นตอนที่ 2: หาค่าของ $z_1 + z_2$

จับส่วนจริงบวกส่วนจริง และส่วนจินตภาพบวกส่วนจินตภาพ:

$$z_1 + z_2 = (3 - 2i) + (1 + 5i)$$

$$z_1 + z_2 = (3 + 1) + (-2i + 5i)$$

$$z_1 + z_2 = 4 + 3i$$

---

### ขั้นตอนที่ 3: ใช้สมบัติของค่าสัมบูรณ์

จากสมบัติของค่าสัมบูรณ์จำนวนเชิงซ้อนที่ว่า $|z^n| = |z|^n$:

$$|(z_1 + z_2)^2| = |z_1 + z_2|^2$$

สูตรการหาขนาดของจำนวนเชิงซ้อน: $|a + bi| = \sqrt{a^2 + b^2}$

ดังนั้น:

$$|4 + 3i| = \sqrt{4^2 + 3^2} = \sqrt{16 + 9} = \sqrt{25} = 5$$

---

### ขั้นตอนที่ 4: ยกกำลังสองตามรูปแบบที่จัดไว้

นำค่าขนาดที่ได้กลับไปยกกำลังสอง:

$$|z_1 + z_2|^2 = 5^2 = 25$$

---

## คำตอบ

$$\boxed{25}$$

---

# เนื้อหาเพื่อศึกษาเพิ่มเติม

## 1) สูตรกำลังสองสมบูรณ์ (Perfect Square Formula)

หนึ่งในเคล็ดลับสำคัญคือการจำสูตรกำลังสองสมบูรณ์:

$$(a+b)^2 = a^2 + 2ab + b^2$$

ในทางกลับกัน: ถ้าเห็น $a^2 + 2ab + b^2$ สามารถรวบเข้าเป็น $(a+b)^2$ ได้

**ประโยชน์**: ช่วยให้การคำนวณง่ายขึ้นมาก โดยไม่ต้องกระจายพจน์ทั้งหมด

ตัวอย่าง:

$$z_1^2 + 2z_1z_2 + z_2^2 = (z_1 + z_2)^2$$

---

## 2) สมบัติของค่าสัมบูรณ์กับเลขชี้กำลัง

สมบัติสำคัญที่ใช้ในโจทย์นี้:

$$|z^n| = |z|^n$$

ความหมาย: แล้วว่าเอาโมดูลัสก่อนหรือหลัง ผลลัพธ์ก็เท่ากัน

**ประโยชน์**: เราสามารถหา $|z|$ ก่อน แล้วค่อยยกกำลัง แทนที่จะคำนวณ $z^n$ แล้วค่อยหา $|z^n|$

ตัวอย่าง:

$$|(z_1+z_2)^2| = |z_1+z_2|^2 = 5^2 = 25$$

แทนที่จะคำนวณ $(z_1+z_2)^2 = (4+3i)^2 = 7+24i$ แล้วค่อยหาโมดูลัส

---

## 3) ค่าสัมบูรณ์ของจำนวนเชิงซ้อน

ถ้า

$$z = a + bi$$

แล้ว

$$|z| = \sqrt{a^2+b^2}$$

และ

$$|z|^2 = a^2 + b^2$$

**ตัวอย่าง**:

$$|4+3i| = \sqrt{4^2+3^2} = \sqrt{25} = 5$$

---

## 4) กลยุทธ์แก้โจทย์ประเภทนี้

**ขั้นตอนสำคัญ**:
1. สำรวจโครงสร้างพหุนาม - มีรูปแบบกำลังสองสมบูรณ์หรือไม่?
2. ถ้าเห็น $a^2 + 2ab + b^2$ ให้รวบเข้า $(a+b)^2$ ทันที
3. ใช้สมบัติ $|z^n| = |z|^n$ เพื่อหาโมดูลัส
4. ลดการคำนวณให้น้อยที่สุด

**ข้อควรระวัง**:
- ไม่ต้องกระจาย $(z_1+z_2)^2$ เพราะมันซับซ้อน
- ใช้ประโยชน์จากสูตร และสมบัติให้มากที่สุด

---

## 5) โจทย์ฝึกเพิ่มเติม

### ข้อฝึก 1
ถ้า $z_1 = 1+2i$ และ $z_2 = 2-i$ จงหา $|z_1^2 + 2z_1z_2 + z_2^2|$

เฉลยย่อ:
$$|(z_1+z_2)^2| = |z_1+z_2|^2 = |3+i|^2 = 9+1 = 10$$

### ข้อฝึก 2
ถ้า $z_1 = 2+i$ และ $z_2 = 1+3i$ จงหา $|(z_1+z_2)^2|$

เฉลยย่อ:
$$|(z_1+z_2)^2| = |3+4i|^2 = 9+16 = 25$$

### ข้อฝึก 3
ถ้า $z = 6+8i$ จงหา $|z^2|$

เฉลยย่อ:
$$|z^2| = |z|^2 = (6^2+8^2) = 100$$
