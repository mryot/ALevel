# โจทย์ลำดับเลขคณิตกับอนุกรมเรขาคณิต

## โจทย์

ให้ $a_1, a_2, a_3, \ldots, a_n, \ldots$ เป็นลำดับเลขคณิต (Arithmetic Sequence) ที่มี $a_6 = -17$ และ $a_{17} = -50$

ถ้า $s$ และ $t$ เป็นจำนวนจริงที่ทำให้

$$3^{a_1} + 3^{a_2} + 3^{a_3} + \cdots + 3^{a_n} + \cdots = \frac{s}{t}$$

โดยที่ $\frac{s}{t}$ เป็นเศษส่วนอย่างต่ำ แล้ว $s + t$ มีค่าเท่าใด

---

## วิธีทำอย่างละเอียด

### ขั้นตอนที่ 1: หาพจน์ทั่วไปของลำดับเลขคณิต

ลำดับเลขคณิตมีรูปแบบทั่วไป:
$$a_n = a_1 + (n-1)d$$

โดยที่ $d$ คือผลต่างร่วม (common difference)

จากเงื่อนไขที่กำหนด:
- $a_6 = a_1 + 5d = -17$ ... (1)
- $a_{17} = a_1 + 16d = -50$ ... (2)

ลบสมการ (1) จากสมการ (2):
$$(a_1 + 16d) - (a_1 + 5d) = -50 - (-17)$$
$$11d = -33$$
$$d = -3$$

แทนค่า $d = -3$ ลงในสมการ (1):
$$a_1 + 5(-3) = -17$$
$$a_1 - 15 = -17$$
$$a_1 = -2$$

ดังนั้น พจน์ทั่วไปคือ:
$$a_n = -2 + (n-1)(-3) = -2 - 3n + 3 = 1 - 3n$$

---

### ขั้นตอนที่ 2: เขียนอนุกรมให้อยู่ในรูปเรขาคณิต

$$\sum_{n=1}^{\infty} 3^{a_n} = \sum_{n=1}^{\infty} 3^{1-3n}$$

แยก $3^{1-3n}$ ออกมา:
$$= \sum_{n=1}^{\infty} 3^1 \cdot 3^{-3n}$$

$$= 3 \sum_{n=1}^{\infty} 3^{-3n}$$

$$= 3 \sum_{n=1}^{\infty} (3^{-3})^n$$

$$= 3 \sum_{n=1}^{\infty} \left(\frac{1}{27}\right)^n$$

---

### ขั้นตอนที่ 3: หาผลบวกของอนุกรมเรขาคณิตอนันต์

นี่คืออนุกรมเรขาคณิตที่มี:
- อัตราส่วนร่วม: $r = \frac{1}{27}$
- เนื่องจาก $|r| = \frac{1}{27} < 1$ อนุกรมจึงลู่เข้า

สูตรผลบวกของอนุกรมเรขาคณิตอนันต์:
$$\sum_{n=1}^{\infty} r^n = \frac{r}{1-r} \quad \text{เมื่อ} \quad |r| < 1$$

ดังนั้น:
$$\sum_{n=1}^{\infty} \left(\frac{1}{27}\right)^n = \frac{\frac{1}{27}}{1 - \frac{1}{27}} = \frac{\frac{1}{27}}{\frac{26}{27}} = \frac{1}{27} \cdot \frac{27}{26} = \frac{1}{26}$$

---

### ขั้นตอนที่ 4: คำนวณค่าของอนุกรมเดิม

$$\sum_{n=1}^{\infty} 3^{a_n} = 3 \cdot \frac{1}{26} = \frac{3}{26}$$

---

### ขั้นตอนที่ 5: หาค่า $s + t$

เศษส่วน $\frac{3}{26}$ เป็นเศษส่วนอย่างต่ำแล้ว เพราะ $\gcd(3, 26) = 1$

ดังนั้น:
- $s = 3$
- $t = 26$
- $s + t = 3 + 26 = 29$

---

## คำตอบ

$$\boxed{29}$$

# เนื้อหาเพื่อศึกษาเพิ่มเติม

## 1) ลำดับเลขคณิต (Arithmetic Sequence)

ลำดับเลขคณิตคือลำดับที่ผลต่างระหว่างพจน์ที่อยู่ติดกันมีค่าคงที่

**สูตรพจน์ทั่วไป**:
$$a_n = a_1 + (n-1)d$$

โดยที่:
- $a_1$ = พจน์แรก
- $d$ = ผลต่างร่วม (common difference)
- $n$ = ลำดับที่ของพจน์

**ตัวอย่าง**: ถ้า $a_1 = 2, d = 3$ แล้ว ลำดับคือ $2, 5, 8, 11, 14, \ldots$

---

## 2) การหาผลต่างร่วม $d$ จากเงื่อนไขสองข้อ

ถ้าทราบ $a_i$ และ $a_j$ (โดยที่ $i \neq j$) เราสามารถหา $d$ ได้:

$$d = \frac{a_j - a_i}{j - i}$$

**ตัวอย่าง**: ถ้า $a_6 = -17$ และ $a_{17} = -50$:
$$d = \frac{-50 - (-17)}{17 - 6} = \frac{-33}{11} = -3$$

---

## 3) อนุกรมเรขาคณิตอนันต์ (Infinite Geometric Series)

**สูตร**: ถ้า $|r| < 1$ แล้ว
$$\sum_{n=1}^{\infty} r^n = \frac{r}{1-r}$$

หรือในรูปทั่วไป:
$$\sum_{n=0}^{\infty} ar^n = \frac{a}{1-r}$$

**เงื่อนไข**: อนุกรมจะลู่เข้า ก็ต่อเมื่อ $|r| < 1$

---

## 4) การแปลงอนุกรมจำนวนเชิงซ้อนเป็นเรขาคณิต

เมื่อเจอปัญหาประเภท $\sum 3^{a_n}$ ที่ $a_n$ เป็นลำดับเลขคณิต:

**ขั้นตอน**:
1. หารูปแบบ $a_n = An + B$
2. เขียน $3^{a_n} = 3^{An+B} = 3^B \cdot (3^A)^n$
3. เห็นว่าเป็นผลคูณ $3^B$ ของอนุกรมเรขาคณิตที่มีอัตราส่วน $r = 3^A$

**ตัวอย่าง**: ถ้า $a_n = 1 - 3n$ แล้ว
$$3^{a_n} = 3^{1-3n} = 3 \cdot (3^{-3})^n = 3 \cdot (1/27)^n$$

---

## 5) เศษส่วนอย่างต่ำ (Fraction in Lowest Terms)

เศษส่วน $\frac{s}{t}$ เป็นเศษส่วนอย่างต่ำ เมื่อ $\gcd(s, t) = 1$

**ตัวอย่าง**:
- $\frac{3}{26}$ เป็นเศษส่วนอย่างต่ำ (เพราะ $\gcd(3, 26) = 1$)
- $\frac{6}{26}$ ไม่เป็นเศษส่วนอย่างต่ำ ต้องลดเป็น $\frac{3}{13}$

---

## 6) กลยุทธ์แก้โจทย์ประเภทนี้

### กลยุทธ์หลัก

1. **ระบุประเภทลำดับ**: ตรวจสอบว่าลำดับเป็นเลขคณิต เรขาคณิต หรือประเภทอื่น

2. **หาพจน์ทั่วไป**: ใช้เงื่อนไขที่กำหนดเพื่อหา $a_n$

3. **แปลงเป็นอนุกรมมาตรฐาน**: พยายามเขียนอนุกรมให้อยู่ในรูปที่รู้สูตร

4. **ตรวจสอบการลู่เข้า**: ตรวจสอบว่า $|r| < 1$ หรือไม่ (สำหรับอนุกรมเรขาคณิต)

5. **ลดเศษส่วน**: ตรวจสอบให้แน่ใจว่าคำตอบอยู่ในรูปเศษส่วนอย่างต่ำ

---

## 7) โจทย์ฝึกเพิ่มเติม

### ข้อฝึก 1
ให้ $b_n$ เป็นลำดับเลขคณิตที่ $b_3 = 5$ และ $b_8 = 20$ 

จงหาผลบวก $\sum_{n=1}^{\infty} 2^{-b_n}$

เฉลยย่อ:
- หา $d = 3$, $b_1 = -1$, $b_n = 3n - 4$
- $\sum_{n=1}^{\infty} 2^{-(3n-4)} = 2^4 \sum_{n=1}^{\infty} (1/8)^n = 16 \cdot \frac{1/8}{7/8} = 2$

### ข้อฝึก 2
ลำดับเลขคณิต $c_n$ มี $c_1 = 4$ และ $d = -2$

จงหาผลบวก $\sum_{n=1}^{\infty} 3^{c_n}$

เฉลยย่อ:
- $c_n = 4 - 2(n-1) = 6 - 2n$
- $\sum_{n=1}^{\infty} 3^{6-2n} = 3^6 \cdot \sum_{n=1}^{\infty} (1/9)^n = 729 \cdot \frac{1/9}{8/9} = \frac{81}{8}$

### ข้อฝึก 3
ให้ $d_n$ เป็นลำดับเลขคณิต ถ้า $\sum_{n=1}^{\infty} 2^{d_n} = \frac{16}{15}$ และ $d_1 = 4$

จงหาผลต่างร่วม $d$

เฉลยย่อ:
- $\sum_{n=1}^{\infty} 2^{d_1 + (n-1)d} = 2^4 \sum_{n=1}^{\infty} (2^d)^{n-1}$
- เพื่อให้ผลบวกเท่ากับ $\frac{16}{15}$ เราต้องให้ $d = -1$
