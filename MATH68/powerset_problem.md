# โจทย์เพาเวอร์เซต

## โจทย์

สำหรับเซต $S$ ใด ๆ ให้ $P(S)$ แทนเพาเวอร์เซตของ $S$

ให้เซต $A = \{\emptyset, \sqrt{2}\}$ แล้วเซต $P(P(A)) \cup P(A)$ มีสมาชิกกี่ตัว

1. 10
2. 16
3. 18
4. 19
5. 20

---

## วิธีทำ

### ขั้นที่ 1: หา $P(A)$

เนื่องจาก $A$ มี 2 สมาชิก ดังนั้น $P(A)$ มี $2^2 = 4$ สมาชิก

$$P(A) = \{\,\emptyset,\ \{\emptyset\},\ \{\sqrt{2}\},\ \{\emptyset, \sqrt{2}\}\,\}$$

### ขั้นที่ 2: หาจำนวนสมาชิกของ $P(P(A))$

เนื่องจาก $P(A)$ มี 4 สมาชิก ดังนั้น $P(P(A))$ มี $2^4 = 16$ สมาชิก

### ขั้นที่ 3: ใช้สูตร $n(X \cup Y) = n(X) + n(Y) - n(X \cap Y)$

ต้องหาว่าสมาชิกของ $P(A)$ ตัวใดบ้างที่อยู่ใน $P(P(A))$ ด้วย (คือเป็น**สับเซต**ของ $P(A)$)

ตรวจทีละตัวของ $P(A)$:

| สมาชิกของ $P(A)$ | เป็นสับเซตของ $P(A)$ หรือไม่ | อยู่ใน $P(P(A))$? |
|---|---|---|
| $\emptyset$ | เป็นสับเซตของทุกเซต | ✓ |
| $\{\emptyset\}$ | $\emptyset \in P(A)$ ✓ | ✓ |
| $\{\sqrt{2}\}$ | $\sqrt{2} \notin P(A)$ ✗ | ✗ |
| $\{\emptyset, \sqrt{2}\}$ | $\sqrt{2} \notin P(A)$ ✗ | ✗ |

ดังนั้น $P(P(A)) \cap P(A)$ มี 2 สมาชิก

### ขั้นที่ 4: คำนวณคำตอบ

$$n\bigl(P(P(A)) \cup P(A)\bigr) = n(P(P(A))) + n(P(A)) - n(P(P(A)) \cap P(A))$$

$$= 16 + 4 - 2 = 18$$

---

## คำตอบ

**ข้อ 3. 18**

# เนื้อหาที่ต้องศึกษาเพิ่มเติม

## 1. เซต (Set) เบื้องต้น

**เซต** คือ กลุ่มของสิ่งของ (เรียกว่า **สมาชิก** หรือ element) ที่ระบุได้ชัดเจน

### สัญลักษณ์ที่สำคัญ

| สัญลักษณ์ | ความหมาย |
|---|---|
| $a \in A$ | $a$ เป็นสมาชิกของเซต $A$ |
| $a \notin A$ | $a$ ไม่เป็นสมาชิกของเซต $A$ |
| $\emptyset$ หรือ $\{\}$ | เซตว่าง (ไม่มีสมาชิก) |
| $n(A)$ | จำนวนสมาชิกของเซต $A$ |
| $A \subseteq B$ | $A$ เป็นสับเซตของ $B$ |
| $A \subset B$ | $A$ เป็นสับเซตแท้ของ $B$ |

### ข้อควรระวัง: $\emptyset$ กับ $\{\emptyset\}$

- $\emptyset$ คือ เซตว่าง มีสมาชิก **0 ตัว**
- $\{\emptyset\}$ คือ เซตที่มีเซตว่างเป็นสมาชิก มีสมาชิก **1 ตัว**

ดังนั้น $\emptyset \neq \{\emptyset\}$

---

## 2. สับเซต (Subset)

**นิยาม:** $A \subseteq B$ ก็ต่อเมื่อ สมาชิกทุกตัวของ $A$ เป็นสมาชิกของ $B$

### สมบัติสำคัญ

1. $\emptyset \subseteq A$ เสมอ (เซตว่างเป็นสับเซตของทุกเซต)
2. $A \subseteq A$ เสมอ (ทุกเซตเป็นสับเซตของตัวเอง)
3. ถ้า $n(A) = n$ แล้ว $A$ มีสับเซตทั้งหมด $2^n$ ตัว
4. ถ้า $n(A) = n$ แล้ว $A$ มีสับเซตแท้ $2^n - 1$ ตัว

### ตัวอย่าง

ให้ $A = \{1, 2\}$ สับเซตของ $A$ ได้แก่:
$$\emptyset,\ \{1\},\ \{2\},\ \{1, 2\}$$

รวม $2^2 = 4$ ตัว

---

## 3. เพาเวอร์เซต (Power Set)

**นิยาม:** เพาเวอร์เซตของ $A$ เขียนแทนด้วย $P(A)$ คือ เซตของสับเซตทั้งหมดของ $A$

$$P(A) = \{\,X \mid X \subseteq A\,\}$$

### สมบัติสำคัญ

1. $n(P(A)) = 2^{n(A)}$
2. $\emptyset \in P(A)$ เสมอ
3. $A \in P(A)$ เสมอ
4. $P(\emptyset) = \{\emptyset\}$ (มีสมาชิก 1 ตัว)
5. $P(P(\emptyset)) = \{\emptyset, \{\emptyset\}\}$ (มีสมาชิก 2 ตัว)

### ความแตกต่างระหว่าง $\in$ และ $\subseteq$

ระวัง! สมาชิกของ $P(A)$ ทุกตัวเป็นสับเซตของ $A$

- $x \in P(A) \iff x \subseteq A$
- ถ้า $a \in A$ แล้ว $\{a\} \in P(A)$ (ไม่ใช่ $a \in P(A)$)

### ตัวอย่าง

ให้ $A = \{1, 2, 3\}$ จะได้
$$P(A) = \{\emptyset, \{1\}, \{2\}, \{3\}, \{1,2\}, \{1,3\}, \{2,3\}, \{1,2,3\}\}$$

มี $2^3 = 8$ สมาชิก

---

## 4. การดำเนินการของเซต

### ยูเนียน (Union): $A \cup B$
$$A \cup B = \{\,x \mid x \in A \text{ หรือ } x \in B\,\}$$

### อินเตอร์เซกชัน (Intersection): $A \cap B$
$$A \cap B = \{\,x \mid x \in A \text{ และ } x \in B\,\}$$

### ผลต่าง (Difference): $A - B$
$$A - B = \{\,x \mid x \in A \text{ และ } x \notin B\,\}$$

### คอมพลีเมนต์ (Complement): $A'$
$$A' = \{\,x \mid x \in U \text{ และ } x \notin A\,\}$$

โดย $U$ คือเอกภพสัมพัทธ์ (Universal set)

---

## 5. สูตรนับจำนวนสมาชิก (Inclusion-Exclusion Principle)

### สำหรับ 2 เซต
$$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$

### สำหรับ 3 เซต
$$n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(A \cap C) - n(B \cap C) + n(A \cap B \cap C)$$

**สูตรนี้ใช้แก้โจทย์ข้างต้นโดยตรง**

---

## 6. สมบัติของเพาเวอร์เซตกับการดำเนินการ

1. $P(A \cap B) = P(A) \cap P(B)$
2. $P(A) \cup P(B) \subseteq P(A \cup B)$ (แต่ไม่จำเป็นต้องเท่ากัน)
3. ถ้า $A \subseteq B$ แล้ว $P(A) \subseteq P(B)$

### ตัวอย่างที่แสดงว่า $P(A) \cup P(B) \neq P(A \cup B)$

ให้ $A = \{1\}, B = \{2\}$
- $P(A) = \{\emptyset, \{1\}\}$
- $P(B) = \{\emptyset, \{2\}\}$
- $P(A) \cup P(B) = \{\emptyset, \{1\}, \{2\}\}$ มี 3 ตัว
- $P(A \cup B) = P(\{1,2\}) = \{\emptyset, \{1\}, \{2\}, \{1,2\}\}$ มี 4 ตัว

---

## 7. เทคนิคการทำโจทย์เพาเวอร์เซตซ้อน

เมื่อเจอ $P(P(A))$ ให้คิดเป็นลำดับขั้น:

1. หา $A$ ก่อน → นับจำนวนสมาชิก
2. หา $P(A)$ → จำนวนสมาชิก = $2^{n(A)}$
3. หา $P(P(A))$ → จำนวนสมาชิก = $2^{n(P(A))} = 2^{2^{n(A)}}$

### ตารางสรุป

| $n(A)$ | $n(P(A))$ | $n(P(P(A)))$ |
|---|---|---|
| 0 | 1 | 2 |
| 1 | 2 | 4 |
| 2 | 4 | 16 |
| 3 | 8 | 256 |
| 4 | 16 | 65536 |

### เทคนิคหา $P(P(A)) \cap P(A)$

สมาชิก $X \in P(P(A)) \cap P(A)$ ก็ต่อเมื่อ
- $X \subseteq P(A)$ (เป็นสมาชิกของ $P(P(A))$)
- $X \subseteq A$ (เป็นสมาชิกของ $P(A)$)

นั่นคือ $X \subseteq A \cap P(A)$

**ตรวจสอบกับโจทย์:** $A = \{\emptyset, \sqrt{2}\}$ และ $P(A) = \{\emptyset, \{\emptyset\}, \{\sqrt{2}\}, \{\emptyset, \sqrt{2}\}\}$

$A \cap P(A) = \{\emptyset\}$ (มีเพียง $\emptyset$ ที่อยู่ทั้งใน $A$ และใน $P(A)$)

ดังนั้น $n(P(P(A)) \cap P(A)) = 2^{n(A \cap P(A))} = 2^1 = 2$ ✓ (ตรงกับที่คำนวณในโจทย์)

---

## 8. โจทย์ฝึกเพิ่มเติม

1. ให้ $A = \{1, \{1\}\}$ จงหาจำนวนสมาชิกของ $P(A)$ และ $P(P(A)) \cap P(A)$
2. ให้ $B = \{\emptyset, \{\emptyset\}\}$ จงหา $P(B)$ และจำนวนสมาชิก
3. ให้ $C = \{a, b, c\}$ จงหาจำนวนสับเซตของ $C$ ที่มีสมาชิก 2 ตัว
4. พิสูจน์ว่า $P(A \cap B) = P(A) \cap P(B)$
