# โจทย์เรขาคณิตวิเคราะห์: ความสูงของสามเหลี่ยมจากพิกัด

## โจทย์

ถ้า $ABC$ เป็นรูปสามเหลี่ยมที่มีพิกัดของ $A, B, C$ คือ $(7,6), (1,2), (1,-2)$ ตามลำดับ
แล้วส่วนสูงของรูปสามเหลี่ยม $ABC$ จากจุด $B$ ไปยังฐาน $AC$ ยาวกี่หน่วย

ตัวเลือก:

1. 0.8
2. 1.6
3. 2.4
4. 4
5. 4.8

---

## วิธีทำอย่างละเอียด (วิธีที่ 1: ใช้พื้นที่สามเหลี่ยม)

โจทย์ถาม "ส่วนสูงจาก $B$ ลงฐาน $AC$" ซึ่งใช้สูตรพื้นที่ได้ตรงที่สุด:

$$\text{พื้นที่สามเหลี่ยม} = \frac{1}{2}(\text{ฐาน})(\text{สูง})$$

ดังนั้น

$$h_B = \frac{2\times \text{พื้นที่}}{|AC|}$$

โดย $h_B$ คือส่วนสูงจากจุด $B$ ลงเส้นฐาน $AC$

### ขั้นที่ 1: หาเวกเตอร์สำคัญ

ให้
$$A(7,6),\ B(1,2),\ C(1,-2)$$

เวกเตอร์

$$\vec{AB} = B-A = (1-7,\ 2-6)=(-6,-4)$$
$$\vec{AC} = C-A = (1-7,\ -2-6)=(-6,-8)$$

### ขั้นที่ 2: หาพื้นที่สามเหลี่ยมจาก cross product (ใน 2D)

ขนาดของ cross product ใน 2 มิติ:

$$|\vec{AB}\times\vec{AC}| = |x_1y_2 - y_1x_2|$$

แทนค่า:

$$|\vec{AB}\times\vec{AC}| = |(-6)(-8)-(-4)(-6)| = |48-24| = 24$$

และ

$$\text{พื้นที่สามเหลี่ยม} = \frac{1}{2}|\vec{AB}\times\vec{AC}| = \frac{1}{2}(24)=12$$

### ขั้นที่ 3: หา长度ของฐาน $AC$

$$|AC| = \sqrt{(1-7)^2+(-2-6)^2} = \sqrt{36+64} = \sqrt{100}=10$$

### ขั้นที่ 4: หาส่วนสูงจาก $B$ ไปฐาน $AC$

$$h_B = \frac{2\times 12}{10} = \frac{24}{10}=2.4$$

ดังนั้นส่วนสูงเท่ากับ

$$\boxed{2.4}$$

## วิธีตรวจสอบอีกแบบ (วิธีที่ 2: ระยะจากจุดถึงเส้นตรง)

### ขั้นที่ 1: หาสมการเส้นตรง $AC$

ความชันของ $AC$:

$$m = \frac{-2-6}{1-7} = \frac{-8}{-6}=\frac{4}{3}$$

ใช้จุด $A(7,6)$:

$$y-6 = \frac{4}{3}(x-7)$$

จัดรูปเป็นมาตรฐาน:

$$3y-18 = 4x-28$$
$$4x-3y-10=0$$

### ขั้นที่ 2: ใช้สูตรระยะจากจุดถึงเส้นตรง

ระยะจากจุด $(x_0,y_0)$ ถึงเส้น $Ax+By+C=0$ คือ

$$d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$$

แทนจุด $B(1,2)$ และเส้น $4x-3y-10=0$:

$$d = \frac{|4(1)-3(2)-10|}{\sqrt{4^2+(-3)^2}} = \frac{|4-6-10|}{\sqrt{25}} = \frac{12}{5}=2.4$$

ตรงกับวิธีแรก ✓

# เนื้อหาเพื่อศึกษาเพิ่มเติม

## 1. สูตรพื้นฐานที่ควรจำ

### 1.1 ระยะระหว่างสองจุด

ถ้า $P(x_1,y_1),Q(x_2,y_2)$

$$PQ = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$

### 1.2 พื้นที่สามเหลี่ยมจากพิกัด

ถ้า $A(x_1,y_1),B(x_2,y_2),C(x_3,y_3)$

$$\text{พื้นที่} = \frac{1}{2}\left|x_1(y_2-y_3)+x_2(y_3-y_1)+x_3(y_1-y_2)\right|$$

### 1.3 ความสัมพันธ์พื้นที่-ฐาน-สูง

$$\text{พื้นที่} = \frac{1}{2}(\text{ฐาน})(\text{สูง})$$

ดังนั้น

$$\text{สูง} = \frac{2\times\text{พื้นที่}}{\text{ฐาน}}$$

### 1.4 ระยะจากจุดถึงเส้นตรง

เส้นตรง $Ax+By+C=0$ และจุด $(x_0,y_0)$:

$$d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$$

## 2. วิธีเลือกแนวทางให้เร็วในข้อสอบ

1. ถ้าโจทย์ให้พิกัดครบ 3 จุดและถาม "ส่วนสูง" ให้คิดสูตรพื้นที่ก่อน
2. ถ้าหาเส้นฐานได้ง่าย ใช้สูตรระยะจากจุดถึงเส้นตรงจะตรงมาก

## 3. โจทย์ฝึกเพิ่มเติม

### ข้อฝึก 1

ให้ $A(0,0),B(4,0),C(1,3)$ จงหาส่วนสูงจาก $C$ ลงฐาน $AB$

แนวคิดย่อ: ฐาน $AB=4$, พื้นที่ $=\frac{1}{2}\times4\times3=6$ จึงได้สูง $=\frac{2\times6}{4}=3$

### ข้อฝึก 2

ให้ $A(2,1),B(5,5),C(8,1)$ จงหาพื้นที่สามเหลี่ยมและส่วนสูงจาก $B$ ลงฐาน $AC$

แนวคิดย่อ: ฐาน $AC=6$ (เส้นแนวนอน), ความสูงจาก $B$ ถึงเส้น $y=1$ คือ 4, พื้นที่ $=\frac{1}{2}\times6\times4=12$

### ข้อฝึก 3

ให้เส้นตรง $3x+4y-20=0$ และจุด $P(2,1)$ จงหาระยะจาก $P$ ถึงเส้นตรง

แนวคิดย่อ:
$$d=\frac{|3(2)+4(1)-20|}{\sqrt{3^2+4^2}}=\frac{10}{5}=2$$
