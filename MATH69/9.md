### **เฉลยละเอียด ข้อ 9**

**โจทย์:** ค่าของ
$$\sin\dfrac{17\pi}{18} + \cos\dfrac{13\pi}{18} + \sin\dfrac{7\pi}{9} - \sin\dfrac{\pi}{18}$$

---

**ขั้นตอนที่ 1: แปลงทุกพจน์ให้อยู่ในรูปง่าย ๆ**

แปลงเป็นหน่วย $\dfrac{\pi}{18}$ เพื่อเทียบกันได้ง่าย: $\dfrac{7\pi}{9} = \dfrac{14\pi}{18}$

**พจน์ที่ 1: $\sin\dfrac{17\pi}{18}$**

ใช้ $\sin(\pi - \theta) = \sin\theta$:
$$\sin\dfrac{17\pi}{18} = \sin\!\left(\pi - \dfrac{\pi}{18}\right) = \sin\dfrac{\pi}{18}$$

**พจน์ที่ 2: $\cos\dfrac{13\pi}{18}$**

ใช้ $\cos(\pi - \theta) = -\cos\theta$:
$$\cos\dfrac{13\pi}{18} = \cos\!\left(\pi - \dfrac{5\pi}{18}\right) = -\cos\dfrac{5\pi}{18}$$

แปลงต่อ ($\cos\theta = \sin(\pi/2 - \theta)$): $\dfrac{\pi}{2} - \dfrac{5\pi}{18} = \dfrac{9\pi - 5\pi}{18} = \dfrac{4\pi}{18}$
$$-\cos\dfrac{5\pi}{18} = -\sin\dfrac{4\pi}{18}$$

**พจน์ที่ 3: $\sin\dfrac{14\pi}{18}$**

ใช้ $\sin(\pi - \theta) = \sin\theta$:
$$\sin\dfrac{14\pi}{18} = \sin\!\left(\pi - \dfrac{14\pi}{18}\right) = \sin\dfrac{4\pi}{18}$$

**พจน์ที่ 4:** คงเดิม $-\sin\dfrac{\pi}{18}$

**ขั้นตอนที่ 2: รวมผล**

$$\sin\dfrac{\pi}{18} - \sin\dfrac{4\pi}{18} + \sin\dfrac{4\pi}{18} - \sin\dfrac{\pi}{18} = 0$$

**ตอบ:** ข้อ 2. $0$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม: สูตรลดทอนมุม (Reduction Formulas)**

**1) ในมุมที่อยู่ในจตุภาคที่ 2 ($\pi/2 < \theta < \pi$)**

| ฟังก์ชัน | สูตร |
|---|---|
| $\sin(\pi - \theta)$ | $\sin\theta$ |
| $\cos(\pi - \theta)$ | $-\cos\theta$ |
| $\tan(\pi - \theta)$ | $-\tan\theta$ |

**2) จตุภาคที่ 3 ($\pi < \theta < 3\pi/2$)**

| ฟังก์ชัน | สูตร |
|---|---|
| $\sin(\pi + \theta)$ | $-\sin\theta$ |
| $\cos(\pi + \theta)$ | $-\cos\theta$ |
| $\tan(\pi + \theta)$ | $\tan\theta$ |

**3) จตุภาคที่ 4 ($3\pi/2 < \theta < 2\pi$)**

| ฟังก์ชัน | สูตร |
|---|---|
| $\sin(2\pi - \theta)$ | $-\sin\theta$ |
| $\cos(2\pi - \theta)$ | $\cos\theta$ |

**4) Cofunction (เปลี่ยน sin↔cos)**

$$\sin\!\left(\dfrac{\pi}{2} - \theta\right) = \cos\theta, \quad \cos\!\left(\dfrac{\pi}{2} - \theta\right) = \sin\theta$$
$$\sin\!\left(\dfrac{\pi}{2} + \theta\right) = \cos\theta, \quad \cos\!\left(\dfrac{\pi}{2} + \theta\right) = -\sin\theta$$

---

### **กลยุทธ์แก้โจทย์ลดทอนมุม**

1. **หาตัวร่วม:** แปลงทุกมุมให้มีหน่วยเดียวกัน (เช่น $\pi/18$ ทั้งหมด)
2. **ตรวจจตุภาค:** ของมุมจริงเพื่อใช้สูตรลดทอนถูกต้อง
3. **จับคู่:** มองหาคู่ที่ "หักล้างกัน" → คำตอบมักเป็น 0
4. **ใช้ cofunction:** เปลี่ยน $\cos$ เป็น $\sin$ (หรือกลับกัน) เพื่อให้รวมพจน์ได้

**กับดักที่พบบ่อย:**
- ลืมเครื่องหมายลบเมื่อใช้ $\cos(\pi - \theta) = -\cos\theta$
- คำนวณจตุภาคผิด → ใช้สูตรผิดด้าน

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** จงหาค่าของ $\sin\dfrac{5\pi}{6} + \cos\dfrac{2\pi}{3} - \cos\dfrac{\pi}{3}$

**เฉลย:**
- $\sin(5\pi/6) = \sin(\pi - \pi/6) = \sin(\pi/6) = 1/2$
- $\cos(2\pi/3) = \cos(\pi - \pi/3) = -\cos(\pi/3) = -1/2$
- $\cos(\pi/3) = 1/2$

รวม: $\dfrac{1}{2} + \left(-\dfrac{1}{2}\right) - \dfrac{1}{2} = -\dfrac{1}{2}$

**ตอบ:** $-\dfrac{1}{2}$
