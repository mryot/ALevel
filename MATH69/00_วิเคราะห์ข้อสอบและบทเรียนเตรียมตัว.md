## 4) บทเรียนที่ 2 — เซตและตรรกศาสตร์

**โจทย์ครอบคลุม:** [2.md](2.md), [3.md](3.md)

### เซต
- $|A \cup B| = |A| + |B| - |A \cap B|$
- 3 เซต: $|A \cup B \cup C| = \sum|A| - \sum|A \cap B| + |A \cap B \cap C|$
- กฎเดอมอร์แกน: $(A \cup B)' = A' \cap B'$, $(A \cap B)' = A' \cup B'$

### ตรรกศาสตร์
| รูป | สมมูล | นิเสธ |
|---|---|---|
| $p \to q$ | $\sim p \lor q$, $\sim q \to \sim p$ | $p \land \sim q$ |
| $p \leftrightarrow q$ | $(p \to q) \land (q \to p)$ | $(p \land \sim q) \lor (q \land \sim p)$ |
| $p \land q$ | — | $\sim p \lor \sim q$ |
| $p \lor q$ | — | $\sim p \land \sim q$ |

**การอ้างเหตุผลสมเหตุสมผล:** สมมติเหตุทั้งหมดเป็นจริง ถ้าผลสรุปเป็นจริงเสมอ → สมเหตุสมผล

### กลยุทธ์
- โจทย์เซต: วาดเวนน์ก่อนเสมอ ใส่ตัวเลขจากในสุด (intersection) ออกข้างนอก
- โจทย์ตรรกศาสตร์: ใช้ตารางค่าความจริงเมื่อสับสน

## 5) บทเรียนที่ 3 — ฟังก์ชัน เอ็กซ์โพเนนเชียล ลอการิทึม

**โจทย์ครอบคลุม:** [4.md](4.md), [5.md](5.md), [6.md](6.md), [7.md](7.md), [8.md](8.md)

### โดเมน-เรนจ์
| ฟังก์ชัน | โดเมน | เรนจ์ |
|---|---|---|
| $\sqrt{x}$ | $x \ge 0$ | $y \ge 0$ |
| $\dfrac{1}{x}$ | $x \ne 0$ | $y \ne 0$ |
| $\log x$ | $x > 0$ | $\mathbb{R}$ |
| $a^x$ | $\mathbb{R}$ | $y > 0$ |
| $\sin x, \cos x$ | $\mathbb{R}$ | $[-1, 1]$ |

### การแปลงกราฟ $y = af(b(x-h)) + k$
- $h > 0$: เลื่อนขวา / $h < 0$: เลื่อนซ้าย
- $k > 0$: เลื่อนขึ้น / $k < 0$: เลื่อนลง
- $|a| > 1$: ยืดแนวตั้ง / $|b| > 1$: บีบแนวนอน
- $a < 0$: สะท้อนแกน $x$ / $b < 0$: สะท้อนแกน $y$

**สำคัญ:** $g(x) = f(x+a)+b$ คือเลื่อน "ซ้าย $a$ หน่วย" และ "ขึ้น $b$ หน่วย" (เปรียบเทียบจุดยอด!)

### ฟังก์ชันกำลังสอง $f(x) = ax^2 + bx + c$
- จุดยอด: $x = -\dfrac{b}{2a}$, $f(x) = c - \dfrac{b^2}{4a}$
- $a > 0$: เปิดขึ้น (มี min) / $a < 0$: เปิดลง (มี max)
- รูปสมบูรณ์: $f(x) = a(x-h)^2 + k$

### เอ็กซ์โพเนนเชียล-ลอการิทึม
| สูตร | หมายเหตุ |
|---|---|
| $a^m \cdot a^n = a^{m+n}$ | คูณ → บวกเลขชี้กำลัง |
| $(a^m)^n = a^{mn}$ | ยกกำลังซ้อน → คูณ |
| $\log(MN) = \log M + \log N$ | log ของผลคูณ |
| $\log M^n = n\log M$ | นำเลขชี้กำลังออกหน้า |
| $\log_a b = \dfrac{\log b}{\log a}$ | เปลี่ยนฐาน |
| $a^{\log_a x} = x$ | สมบัติผกผัน |

**เทคนิค Vieta แก้สมการกำลังสองที่มี log:**  
ถ้า $x^2 + bx + c = 0$ มีราก $x_1, x_2$ แล้ว $x_1 + x_2 = -b$, $x_1 x_2 = c$

### อสมการ log/exp
- เปลี่ยนฐานเท่ากันก่อน, ระวังฐาน $0 < a < 1$ → เครื่องหมายอสมการ **กลับ**
- log: ต้องตรวจ argument > 0 เสมอ

## 6) บทเรียนที่ 4 — ตรีโกณมิติ

**โจทย์ครอบคลุม:** [9.md](9.md), [10.md](10.md)

### เอกลักษณ์พื้นฐาน
$$\sin^2\theta + \cos^2\theta = 1, \quad 1+\tan^2\theta = \sec^2\theta, \quad 1+\cot^2\theta = \csc^2\theta$$

### สูตรมุมสองเท่า / ครึ่ง
- $\sin 2\theta = 2\sin\theta\cos\theta$
- $\cos 2\theta = \cos^2\theta - \sin^2\theta = 1 - 2\sin^2\theta = 2\cos^2\theta - 1$

### กฎไซน์ / กฎโคไซน์
- กฎไซน์: $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$
- กฎโคไซน์: $c^2 = a^2 + b^2 - 2ab\cos C$
- พื้นที่: $\dfrac{1}{2}ab\sin C$ หรือสูตรเฮรอน

### กลยุทธ์
- โจทย์ "พิสูจน์เอกลักษณ์": จัดข้างหนึ่งให้เท่ากับอีกข้าง โดยใช้สูตรพื้นฐานก่อน
- โจทย์สามเหลี่ยม: ต้องการ "มุม-ด้าน-มุม" หรือ "ด้าน-ด้าน" → ใช้กฎไซน์ ; ต้องการ "มุมจากด้านทั้งหมด" → กฎโคไซน์

## 7) บทเรียนที่ 5 — เมทริกซ์และจำนวนเชิงซ้อน

**โจทย์ครอบคลุม:** [11.md](11.md), [12.md](12.md), [13.md](13.md)

### เมทริกซ์
- $\det(AB) = \det(A) \cdot \det(B)$
- $\det(A^{-1}) = \dfrac{1}{\det(A)}$
- $\det(kA) = k^n \det(A)$ (เมทริกซ์ขนาด $n \times n$)
- $\det(A^T) = \det(A)$

### จำนวนเชิงซ้อน
| รูป | สูตร |
|---|---|
| คาร์ทีเซียน | $z = a + bi$ |
| โพลาร์ | $z = r(\cos\theta + i\sin\theta) = re^{i\theta}$ |
| โมดูลัส | $|z| = \sqrt{a^2 + b^2}$ |
| คอนจูเกต | $\bar z = a - bi$, $z\bar z = |z|^2$ |
| คูณ | $r_1 r_2 [\cos(\theta_1+\theta_2) + i\sin(\theta_1+\theta_2)]$ |
| ทฤษฎีบทเดอมัวฟ์ | $z^n = r^n(\cos n\theta + i\sin n\theta)$ |

### รากที่ $n$ ของ 1
$$z_k = \cos\dfrac{2\pi k}{n} + i\sin\dfrac{2\pi k}{n}, \quad k = 0, 1, \ldots, n-1$$
รากทั้งหมดอยู่บนวงกลมหนึ่งหน่วย กระจายเท่ากัน ผลรวมราก $= 0$ (เมื่อ $n \ge 2$)

### โจทย์เรขาคณิตเชิงซ้อน
- $|z - z_0| = r$ → วงกลมจุดศูนย์กลาง $z_0$ รัศมี $r$
- $|z - a| = |z - b|$ → เส้นแบ่งครึ่งตั้งฉาก $\overline{ab}$
- $\text{Re}(z) < c$ → ฝั่งซ้ายของเส้น $x = c$

## 8) บทเรียนที่ 6 — ลำดับ อนุกรม และแคลคูลัส

**โจทย์ครอบคลุม:** [14.md](14.md), [24.md](24.md), [25.md](25.md), [27.md](27.md), [30.md](30.md)

### ลำดับและอนุกรม
| ประเภท | พจน์ที่ $n$ | ผลรวม |
|---|---|---|
| เลขคณิต | $a_n = a_1 + (n-1)d$ | $S_n = \dfrac{n}{2}(a_1 + a_n)$ |
| เรขาคณิต | $a_n = a_1 r^{n-1}$ | $S_n = a_1 \dfrac{1-r^n}{1-r}$ |
| เรขาคณิตอนันต์ ($|r|<1$) | — | $S_\infty = \dfrac{a_1}{1-r}$ |

### ลิมิตและความต่อเนื่อง
- $\displaystyle\lim_{x\to a}f(x)$ มีค่า $\iff$ ลิมิตซ้าย = ลิมิตขวา
- ต่อเนื่องที่ $x=a$ $\iff$ $\displaystyle\lim_{x\to a}f(x) = f(a)$
- **อ่านกราฟ:** จุดทึบ $\bullet$ = ค่าจริงของฟังก์ชัน ; จุดเปิด $\circ$ = ไม่ใช่ค่าจริง

### อนุพันธ์
- $\dfrac{d}{dx}x^n = nx^{n-1}$
- กฎผลคูณ: $(uv)' = u'v + uv'$
- กฎลูกโซ่: $\dfrac{dy}{dx} = \dfrac{dy}{du} \cdot \dfrac{du}{dx}$
- ความชันเส้นสัมผัส = $f'(x_0)$
- $f'>0$: เพิ่ม / $f'<0$: ลด / $f'=0$: จุดวิกฤต
- **อ่านความชันจากกราฟ:** ยิ่งกราฟชัน → $|f'|$ ยิ่งมาก ; แนวนอน → $f' = 0$

### ปริพันธ์
$$\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C \;(n\ne -1), \quad \int_a^b f(x)\,dx = F(b)-F(a)$$
- พื้นที่ใต้กราฟ: $\int_a^b f(x)\,dx$ (เหนือแกน $x$)
- พื้นที่ระหว่างสองกราฟ: $\int_a^b [f(x)-g(x)]\,dx$

## 9) บทเรียนที่ 7 — เรขาคณิตวิเคราะห์ ภาคตัดกรวย และเวกเตอร์

**โจทย์ครอบคลุม:** [15.md](15.md), [16.md](16.md), [17.md](17.md), [18.md](18.md), [26.md](26.md)

### ภาคตัดกรวย
| กราฟ | สมการมาตรฐาน |
|---|---|
| วงกลม | $(x-h)^2+(y-k)^2 = r^2$ |
| พาราโบลา | $(y-k)^2 = 4p(x-h)$ หรือ $(x-h)^2 = 4p(y-k)$ |
| วงรี | $\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = 1$ |
| ไฮเพอร์โบลา | $\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = 1$ |

**จำแนกจากสมการทั่วไป $Ax^2 + Cy^2 + \ldots = 0$:**
- $A = C$ → วงกลม
- $AC > 0$, $A \ne C$ → วงรี
- $AC = 0$ (เหลือแค่ตัวเดียวยกกำลังสอง) → พาราโบลา
- $AC < 0$ → ไฮเพอร์โบลา

### เวกเตอร์ 3 มิติ
| สูตร | ความหมาย |
|---|---|
| $\vec a \cdot \vec b = a_1 b_1 + a_2 b_2 + a_3 b_3$ | dot product |
| $\vec a \cdot \vec b = \|\vec a\|\|\vec b\|\cos\theta$ | หา angle |
| $\vec a \perp \vec b \iff \vec a \cdot \vec b = 0$ | ตั้งฉาก |
| $\vec a \times \vec b$ | cross product (ตั้งฉากกับทั้ง $\vec a$ และ $\vec b$) |
| $\|\vec a \times \vec b\|$ | พื้นที่สี่เหลี่ยมด้านขนาน |
| $\dfrac{1}{2}\|\vec{AB} \times \vec{AC}\|$ | พื้นที่สามเหลี่ยม |

### เรขาคณิตวิเคราะห์ระนาบ
- ระยะทาง: $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
- ความชัน: $m = \dfrac{y_2-y_1}{x_2-x_1}$
- เส้นขนาน: $m_1 = m_2$ ; ตั้งฉาก: $m_1 m_2 = -1$
- ระยะจากจุดถึงเส้น $Ax+By+C=0$: $\dfrac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$

## 10) บทเรียนที่ 8 — ความน่าจะเป็นและสถิติ

**โจทย์ครอบคลุม:** [19.md](19.md), [20.md](20.md), [21.md](21.md), [22.md](22.md), [23.md](23.md), [28.md](28.md), [29.md](29.md)

### การนับ
| สถานการณ์ | สูตร |
|---|---|
| เรียง $n$ สิ่งต่างกัน | $n!$ |
| เลือก $r$ จาก $n$ แบบเรียง | $P(n,r) = \dfrac{n!}{(n-r)!}$ |
| เลือก $r$ จาก $n$ ไม่สนลำดับ | $\binom{n}{r} = \dfrac{n!}{r!(n-r)!}$ |
| เรียงเป็นวงกลม | $(n-1)!$ |
| เรียงของซ้ำ | $\dfrac{n!}{n_1! n_2! \cdots}$ |

### ความน่าจะเป็น
- $P(A) = \dfrac{n(A)}{n(S)}$
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
- $P(A') = 1 - P(A)$
- เหตุการณ์เป็นอิสระ: $P(A \cap B) = P(A) \cdot P(B)$
- เงื่อนไข: $P(B|A) = \dfrac{P(A \cap B)}{P(A)}$

### การแจกแจง
**ทวินาม** $X \sim B(n, p)$
$$P(X = k) = \binom{n}{k}p^k(1-p)^{n-k}, \quad E(X) = np, \; V(X) = np(1-p)$$

**ปกติ** $X \sim N(\mu, \sigma^2)$  
- $Z = \dfrac{X-\mu}{\sigma}$
- 68% อยู่ใน $\mu \pm \sigma$ / 95% อยู่ใน $\mu \pm 2\sigma$ / 99.7% อยู่ใน $\mu \pm 3\sigma$
- $P(Z > 1) \approx 0.1587$, $P(Z > 2) \approx 0.0228$

### สถิติเชิงพรรณนา
| ค่า | สูตร |
|---|---|
| ค่าเฉลี่ย | $\bar x = \dfrac{\sum x_i}{n}$ |
| มัธยฐาน | ค่ากึ่งกลางเมื่อเรียง |
| ฐานนิยม | ค่าที่ปรากฏบ่อยที่สุด |
| ความแปรปรวน | $s^2 = \dfrac{\sum(x_i - \bar x)^2}{n}$ |
| ส่วนเบี่ยงเบนมาตรฐาน | $s = \sqrt{s^2}$ |

**เทคนิค:** $s^2 = \dfrac{\sum x_i^2}{n} - \bar x^2$ (สะดวกกว่า)
