### **เฉลยละเอียด ข้อ 10**

**โจทย์:** มุมของสามเหลี่ยมมีอัตราส่วน $2:3:7$ จงหาอัตราส่วนของความยาวด้านตรงข้าม

---

**ขั้นตอนที่ 1: หาขนาดมุมจริง**

ผลรวมมุมสามเหลี่ยม $= 180° = \pi$ เรเดียน
มุม 3 มุมแบ่งเป็น $2+3+7 = 12$ ส่วน → 1 ส่วน $= 15°$

$$A = 2 \times 15° = 30°, \quad B = 3 \times 15° = 45°, \quad C = 7 \times 15° = 105°$$

**ขั้นตอนที่ 2: ใช้กฎไซน์**

$$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C}$$

อัตราส่วนด้าน $a : b : c = \sin A : \sin B : \sin C = \sin 30° : \sin 45° : \sin 105°$

**ขั้นตอนที่ 3: คำนวณ sin แต่ละมุม**

- $\sin 30° = \dfrac{1}{2}$
- $\sin 45° = \dfrac{\sqrt{2}}{2}$
- $\sin 105° = \sin(60° + 45°)$
  $= \sin 60°\cos 45° + \cos 60°\sin 45°$
  $= \dfrac{\sqrt{3}}{2} \cdot \dfrac{\sqrt{2}}{2} + \dfrac{1}{2} \cdot \dfrac{\sqrt{2}}{2}$
  $= \dfrac{\sqrt{6}}{4} + \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{6} + \sqrt{2}}{4}$

**ขั้นตอนที่ 4: จัดอัตราส่วน**

$$a:b:c = \dfrac{1}{2} : \dfrac{\sqrt{2}}{2} : \dfrac{\sqrt{6}+\sqrt{2}}{4}$$

คูณทุกพจน์ด้วย 4:
$$= 2 : 2\sqrt{2} : \sqrt{6}+\sqrt{2}$$

หารทุกพจน์ด้วย $\sqrt{2}$:
$$= \dfrac{2}{\sqrt{2}} : 2 : \dfrac{\sqrt{6}+\sqrt{2}}{\sqrt{2}} = \sqrt{2} : 2 : \sqrt{3}+1$$

**ตอบ:** ข้อ 1. $\sqrt{2} : 2 : \sqrt{3}+1$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม**

**1) กฎไซน์ (Law of Sines)**

$$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$$

โดย $R$ คือรัศมีวงกลมล้อมรอบ (circumradius)

**2) กฎโคไซน์ (Law of Cosines)**

$$a^2 = b^2 + c^2 - 2bc\cos A$$

**3) สูตรผลคูณมุม**

| สูตร |
|---|
| $\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$ |
| $\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$ |
| $\sin 2A = 2\sin A \cos A$ |
| $\cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A$ |

**4) ค่ามุมพิเศษ**

| $\theta$ | $\sin\theta$ | $\cos\theta$ | $\tan\theta$ |
|---|---|---|---|
| $30°$ | $1/2$ | $\sqrt{3}/2$ | $1/\sqrt{3}$ |
| $45°$ | $\sqrt{2}/2$ | $\sqrt{2}/2$ | $1$ |
| $60°$ | $\sqrt{3}/2$ | $1/2$ | $\sqrt{3}$ |

### **กลยุทธ์แก้โจทย์สามเหลี่ยมที่ให้อัตราส่วนมุม**

1. **แปลงเป็นองศาจริง:** หาว่าแต่ละส่วนกี่องศา
2. **ใช้กฎไซน์:** ด้านอัตราส่วน = $\sin$ ของมุมตรงข้าม
3. **จัดรูปอัตราส่วน:** คูณ/หารด้วยตัวคงที่เพื่อให้สอดคล้องตัวเลือก
4. **แยก $\sin$ มุมไม่มาตรฐาน:** เช่น $\sin 105° = \sin(60° + 45°)$
5. **Rationalize:** หาร $\sqrt{2}$ เพื่อให้รูปสวย

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** สามเหลี่ยมหนึ่งมีมุมในอัตราส่วน $1:2:3$ จงหาอัตราส่วนของด้านตรงข้าม

**เฉลย:**
- มุม: $30°, 60°, 90°$ (สามเหลี่ยมมุมฉาก!)
- ด้าน: $\sin 30° : \sin 60° : \sin 90° = \dfrac{1}{2} : \dfrac{\sqrt{3}}{2} : 1$
- คูณ 2: $1 : \sqrt{3} : 2$

**ตอบ:** $1 : \sqrt{3} : 2$ (สามเหลี่ยม 30-60-90 พื้นฐาน)
