### **เฉลยละเอียด ข้อ 29**

**โจทย์:** ข้อมูล: $-6, x, y, z, 3$ เรียงน้อยไปมาก
ค่าเฉลี่ย $= 0$, ฐานนิยม $= 1$ จงหาความแปรปรวน

---

**ขั้นตอนที่ 1: ใช้เงื่อนไขค่าเฉลี่ย**

$$\dfrac{-6 + x + y + z + 3}{5} = 0$$
$$x + y + z = 3 \quad (*)$$

**ขั้นตอนที่ 2: ใช้เงื่อนไขฐานนิยม $= 1$**

ฐานนิยมต้องเป็นค่าที่ปรากฏ **บ่อยกว่าค่าอื่น**

ข้อมูล $-6$ และ $3$ ปรากฏ 1 ครั้งแล้ว → ค่า 1 ต้องปรากฏ **อย่างน้อย 2 ครั้ง** ในกลุ่ม $\{x, y, z\}$

**ขั้นตอนที่ 3: ลองกรณีต่าง ๆ ของ $x, y, z$**

ต้องเรียงน้อยไปมาก: $-6 \le x \le y \le z \le 3$

**กรณีมี 1 อย่างน้อย 2 ตัวใน $\{x,y,z\}$:**

ลองให้ 2 ตัวเป็น 1:
- $y = z = 1$: จาก (*) $x = 3 - 2 = 1$ → ทั้ง 3 ตัวเป็น 1
- $x = y = 1$: จาก (*) $z = 1$ → ทั้ง 3 ตัวเป็น 1
- $x = z = 1$: ต้อง $y = 1$ → ทั้ง 3 ตัวเป็น 1

→ มีกรณีเดียว: $x = y = z = 1$

**ตรวจสอบ:**
- ข้อมูล: $-6, 1, 1, 1, 3$ ✓ เรียงน้อยไปมาก
- ค่าเฉลี่ย: $(-6+1+1+1+3)/5 = 0$ ✓
- ฐานนิยม: $1$ (ปรากฏ 3 ครั้ง) ✓

**ขั้นตอนที่ 4: คำนวณความแปรปรวน**

ใช้สูตรความแปรปรวนประชากร:
$$\sigma^2 = \dfrac{\sum (x_i - \bar{x})^2}{N} = \dfrac{\sum x_i^2}{N} - \bar{x}^2$$

เนื่องจาก $\bar{x} = 0$:
$$\sigma^2 = \dfrac{\sum x_i^2}{N}$$

$$\sum x_i^2 = (-6)^2 + 1^2 + 1^2 + 1^2 + 3^2 = 36 + 1 + 1 + 1 + 9 = 48$$

$$\sigma^2 = \dfrac{48}{5} = 9.6$$

**ตอบ:** $\boxed{9.6}$

### **เนื้อหาเพื่อศึกษาเพิ่มเติม**

**1) ค่ากลาง (Central Tendency)**

| ค่ากลาง | นิยาม |
|---|---|
| ค่าเฉลี่ย (Mean) | ผลรวมหารจำนวนข้อมูล |
| มัธยฐาน (Median) | ค่ากลางเมื่อเรียง |
| ฐานนิยม (Mode) | ค่าที่ปรากฏบ่อยที่สุด |

**2) ความแปรปรวน (Variance)**

**สำหรับประชากร** ($N$ ตัว):
$$\sigma^2 = \dfrac{\sum_{i=1}^{N}(x_i - \mu)^2}{N}$$

**สำหรับตัวอย่าง** ($n$ ตัว):
$$s^2 = \dfrac{\sum_{i=1}^{n}(x_i - \bar{x})^2}{n-1}$$

> โจทย์นี้ใช้คำว่า "ประชากร" → ใช้ตัวหาร $N = 5$

**3) สูตรลัด**

$$\sigma^2 = \dfrac{\sum x_i^2}{N} - \bar{x}^2$$

ใช้งานง่ายเมื่อ $\bar{x} = 0$: ตัวที่สองหายไป

**4) ส่วนเบี่ยงเบนมาตรฐาน**

$$\sigma = \sqrt{\sigma^2}$$

### **กลยุทธ์แก้โจทย์สถิติเงื่อนไขผสม**

1. **ใช้เงื่อนไขค่าเฉลี่ย** → ได้สมการความสัมพันธ์ของตัวแปร
2. **ใช้เงื่อนไขฐานนิยม** → จำกัดให้บางค่าซ้ำ
3. **ใช้เงื่อนไขอันดับ** (เรียงน้อยไปมาก) → ลำดับของตัวแปร
4. **ลองทุกกรณี** ที่เป็นไปได้ของตัวแปร แล้วเลือกที่สอดคล้องทุกเงื่อนไข
5. **คำนวณความแปรปรวน** ด้วยสูตรที่เหมาะ (สังเกตว่าเป็นประชากรหรือตัวอย่าง)

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** ข้อมูลประชากร 5 ตัว: $1, 3, x, 5, 9$ มีค่าเฉลี่ย $= 4$ จงหาความแปรปรวน

**เฉลย:**
- ผลรวม $= 5 \times 4 = 20$ → $x = 20 - (1+3+5+9) = 2$
- ข้อมูล: $1, 2, 3, 5, 9$
- ความแปรปรวน $\sigma^2 = \dfrac{(1-4)^2 + (2-4)^2 + (3-4)^2 + (5-4)^2 + (9-4)^2}{5}$
- $= \dfrac{9 + 4 + 1 + 1 + 25}{5} = \dfrac{40}{5} = 8$

**ตอบ:** $8$
