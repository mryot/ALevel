## 1. เฉลยวิธีทำอย่างละเอียด

โจทย์ข้อนี้เป็นเรื่อง **"การเลื่อนขนานกราฟ (Graph Translation)"** ของฟังก์ชัน

### **ขั้นตอนที่ 1: หาจุดอ้างอิงที่เด่นชัดจากกราฟ $f$**

สังเกตจุดยอด (Peaks) และจุดต่ำสุด (Valleys) ของกราฟ $f$ ที่ตัดกับเส้นตาราง (Grid) อย่างลงตัว:

* **จุดต่ำสุดฝั่งซ้าย:** $(-2, -2)$
* **จุดสูงสุด (จุดยอด):** $(0, 0)$
* **จุดต่ำสุดฝั่งขวา:** $(4, -4)$

### **ขั้นตอนที่ 2: หาจุดอ้างอิงที่สอดคล้องกันบนกราฟ $g$**

มองหาจุดยอดและจุดต่ำสุดในลักษณะเดียวกันบนกราฟ $g$:

* **จุดต่ำสุดฝั่งซ้าย:** $(0, 1)$
* **จุดสูงสุด (จุดยอด):** $(2, 3)$
* **จุดต่ำสุดฝั่งขวา:** $(6, -1)$

### **ขั้นตอนที่ 3: วิเคราะห์การเลื่อนขนาน (Translation)**

เมื่อเปรียบเทียบจุดต่อจุดจากกราฟ $f$ ไปยังกราฟ $g$:

* **พิจารณาแกน X (แนวนอน):** จากเดิมจุดยอดอยู่ที่ $x = 0$ เลื่อนไปอยู่ที่ $x = 2$ แสดงว่า **เลื่อนไปทางขวา 2 หน่วย**
* **พิจารณาแกน Y (แนวตั้ง):** จากเดิมจุดยอดอยู่ที่ $y = 0$ เลื่อนไปอยู่ที่ $y = 3$ แสดงว่า **เลื่อนขึ้นข้างบน 3 หน่วย**

*(ลองเช็กกับจุดต่ำสุดฝั่งขวา: จาก $(4, -4)$ ไปยัง $(4+2, -4+3) = (6, -1)$ ซึ่งตรงกันพอดี)*

### **ขั้นตอนที่ 4: สมการการเลื่อนขนาน**

จากหลักการเลื่อนขนานฟังก์ชัน:

* เลื่อนขนานไปทางขวา $h$ หน่วย จะได้ $f(x - h)$
* เลื่อนขนานขึ้นบน $k$ หน่วย จะได้ $+ k$

ในข้อนี้เลื่อนขนานไปทางขวา 2 หน่วย และขึ้นบน 3 หน่วย จะได้สมการของ $g(x)$ คือ:


$$g(x) = f(x - 2) + 3$$

### **ขั้นตอนที่ 5: หาค่า $a$ และ $b$ เพื่อหาคำตอบ**

นำสมการที่ได้ไปเปรียบเทียบกับรูปแบบที่โจทย์กำหนด: $g(x) = f(x + a) + b$

* จะได้ $a = -2$
* จะได้ $b = 3$

โจทย์ถามหาค่าของ $2a + 3b$:


$$2a + 3b = 2(-2) + 3(3)$$

$$2a + 3b = -4 + 9 = 5$$

**ตอบ ข้อ 4. 5**

## 2. เนื้อหาเพิ่มเติมเพื่อศึกษา: การแปลงกราฟ (Function Transformations)

เพื่อให้เข้าใจโจทย์ประเภทนี้มากขึ้น มาทบทวนกฎพื้นฐานของการเลื่อนขนานกราฟกันครับ สมมติให้กราฟเริ่มต้นคือ $y = f(x)$ โดยที่ $h > 0$ และ $k > 0$

| การเปลี่ยนแปลงในสมการ | รูปแบบกราฟที่ได้ | ลักษณะการเลื่อนขนาน |
| --- | --- | --- |
| **เปลี่ยนที่ตัวแปร $x$** | $y = f(x - h)$ | เลื่อนขนานไปทาง **ขวา** $h$ หน่วย |
|  | $y = f(x + h)$ | เลื่อนขนานไปทาง **ซ้าย** $h$ หน่วย |
| **เปลี่ยนที่ผลลัพธ์ภายนอก** | $y = f(x) + k$ | เลื่อนขนานขึ้น **บน** $k$ หน่วย |
|  | $y = f(x) - k$ | เลื่อนขนานลง **ล่าง** $k$ หน่วย |

> **ข้อควรระวัง:** การเลื่อนขนานในแนวนอน (แกน X) เครื่องหมายในวงเล็บจะ **ตรงข้าม** กับทิศทางที่เรารู้สึก เช่น $x - 2$ คือเลื่อนไปทางบวก (ขวา) ส่วนการเลื่อนขนานในแนวตั้ง (แกน Y) เครื่องหมายจะเป็นไปตามทิศทางปกติ $+3$ คือเลื่อนขึ้นบน

## 3. กลยุทธ์ในการแก้โจทย์ประเภทนี้

1. **จับตาดู "จุดยอด" หรือ "จุดหักเห":** อย่ามองกราฟภาพรวมทั้งหมด ให้เลือกจุดที่อ่านค่าพิกัด $(x, y)$ ได้ง่ายและชัดเจนที่สุดมา 1-2 จุด
2. **ตั้งสมการเลื่อนขนานด้วยตัวเองก่อน:** คำนวณระยะเลื่อน $\Delta x$ และ $\Delta y$ แล้วเขียนให้อยู่ในรูป $f(x - h) + k$ ให้เสร็จสรรพ
3. **เทียบสัมประสิทธิ์อย่างระมัดระวัง:** เมื่อได้สมการแล้ว ค่อยนำกลับไปเทียบกับสิ่งที่โจทย์ให้มา โดยเฉพาะเครื่องหมายหน้าตัวแปร $a$ และ $b$ เพื่อป้องกันความผิดพลาดเรื่องเครื่องหมายบวก/ลบ

## 4. โจทย์เพิ่มเติมเพื่อฝึกฝนพร้อมเฉลย

### **โจทย์ข้อที่ 1**

ให้กราฟ $y = f(x)$ มีจุดต่ำสุดสัมพัทธ์อยู่ที่พิกัด $(1, -3)$ ถ้านำกราฟนี้ไปเลื่อนขนานจนเกิดเป็นฟังก์ชันใหม่คือ $g(x) = f(x - 4) + 5$ อยากทราบว่าจุดต่ำสุดสัมพัทธ์ของกราฟ $g(x)$ จะเปลี่ยนไปอยู่ที่พิกัดใด

**แนวคิด:**

1. จากสมการ $g(x) = f(x - 4) + 5$ หมายถึงการเลื่อนขนานกราฟ $f(x)$ ไปทาง**ขวา 4 หน่วย** และเลื่อน**ขึ้นบน 5 หน่วย**
2. นำพิกัดจุดเดิม $(1, -3)$ มาคำนวณการเลื่อน:
* พิกัด $x$ ใหม่: $1 + 4 = 5$
* พิกัด $y$ ใหม่: $-3 + 5 = 2$



**ตอบ:** จุดต่ำสุดสัมพัทธ์ของกราฟ $g(x)$ อยู่ที่พิกัด $(5, 2)$

### **โจทย์ข้อที่ 2**

กำหนดฟังก์ชัน $g(x) = f(x + 3) - 1$ ถ้ากราฟของ $g(x)$ มีจุดตัดแกน Y อยู่ที่จุด $(0, 4)$ อยากทราบว่าจุดพิกัดเดิมบนกราฟ $f(x)$ ที่สอดคล้องกันคือจุดใด

**แนวคิด:**

1. โจทย์ข้อนี้เป็นการคิด "ย้อนกลับ" จากกราฟผลลัพธ์ $g(x)$ กลับไปหากราฟตั้งต้น $f(x)$
2. จาก $g(x) = f(x + 3) - 1$ แปลว่า กราฟ $f$ ถูกเลื่อน**ซ้าย 3 หน่วย** และ**ลงล่าง 1 หน่วย** จนกลายเป็นกราฟ $g$
3. ดังนั้น ถ้าจะหาจุดเดิมบนกราฟ $f$ เราต้องทำกระบวนการตรงกันข้ามกับจุดบนกราฟ $g$ ซึ่งก็คือ เลื่อน**ขวา 3 หน่วย** และเลื่อน**ขึ้นบน 1 หน่วย**
4. นำพิกัด $(0, 4)$ ของกราฟ $g$ มาแปลงกลับ:
* พิกัด $x$ เดิม: $0 + 3 = 3$
* พิกัด $y$ เดิม: $4 + 1 = 5$



**ตอบ:** จุดบนกราฟ $f(x)$ ที่สอดคล้องกันคือจุด $(3, 5)$
