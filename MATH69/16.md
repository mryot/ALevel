### **เฉลยละเอียด ข้อ 16**

**โจทย์:** $y^2 - x^2 + 6x - 8 = 0$ เป็นกราฟอะไร

---

**ขั้นตอนที่ 1: จัดกำลังสองสมบูรณ์**

$$y^2 - (x^2 - 6x) - 8 = 0$$
$$y^2 - (x^2 - 6x + 9 - 9) - 8 = 0$$
$$y^2 - (x - 3)^2 + 9 - 8 = 0$$
$$y^2 - (x - 3)^2 + 1 = 0$$
$$y^2 - (x - 3)^2 = -1$$

คูณ $-1$ ทั้งสองข้าง:
$$(x - 3)^2 - y^2 = 1$$

**ขั้นตอนที่ 2: จำแนกชนิดกราฟ**

รูป $\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = 1$ คือ **ไฮเพอร์โบลา** ที่มีแกนตามขวาง **ขนานกับแกน $X$** (เปิดซ้าย-ขวา)

ในที่นี้: $h = 3, k = 0, a^2 = 1, b^2 = 1$

**ตอบ:** ข้อ 4. ไฮเพอร์โบลาที่มีแกนตามขวางขนานกับแกน $X$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม: ภาคตัดกรวย (Conic Sections)**

**1) สมการมาตรฐาน**

| ชนิด | สมการ |
|---|---|
| วงกลม | $(x-h)^2 + (y-k)^2 = r^2$ |
| วงรี (แกน X) | $\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = 1$, $a > b$ |
| วงรี (แกน Y) | $\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = 1$, $b > a$ |
| พาราโบลา (เปิดขึ้น) | $(x-h)^2 = 4p(y-k)$ |
| ไฮเพอร์โบลา (เปิด X) | $\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = 1$ |
| ไฮเพอร์โบลา (เปิด Y) | $\dfrac{(y-k)^2}{a^2} - \dfrac{(x-h)^2}{b^2} = 1$ |

**2) สมการรูปทั่วไป**

$$Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0$$

ตัวจำแนก (Discriminant) $= B^2 - 4AC$:
- $< 0$ : วงรี (รวมวงกลมเมื่อ $A = C, B = 0$)
- $= 0$ : พาราโบลา
- $> 0$ : ไฮเพอร์โบลา

**3) ตรวจเร็วเมื่อ $B = 0$**

ดูเครื่องหมายของ $A$ และ $C$:
- $A = C$ : วงกลม
- $A, C$ เครื่องหมายเดียวกัน แต่ไม่เท่า : วงรี
- $A, C$ ต่างเครื่องหมาย : ไฮเพอร์โบลา
- ตัวใดตัวหนึ่งเป็น 0 : พาราโบลา

ในโจทย์: $-x^2 + y^2 + \ldots$ → $A = -1, C = 1$ ต่างเครื่องหมาย → **ไฮเพอร์โบลา**

---

### **กลยุทธ์แก้โจทย์จำแนกกราฟ**

1. **สังเกตเครื่องหมายของ $x^2$ และ $y^2$ ก่อน**
   - เหมือนกัน → วงกลม/วงรี
   - ต่างกัน → ไฮเพอร์โบลา
   - ตัวเดียว → พาราโบลา
2. **จัดกำลังสองสมบูรณ์** เพื่อหา center และพารามิเตอร์
3. **คูณ $-1$ ทั้งสมการ** ถ้าด้านขวาเป็นค่าลบ เพื่อให้เห็นรูปมาตรฐาน
4. **ดูตัวที่เป็นบวก:** สำหรับไฮเพอร์โบลา $\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = 1$ พจน์บวกเป็น $x$ → แกนตามขวางขนานแกน $X$

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** $4x^2 + 9y^2 - 16x + 18y - 11 = 0$ เป็นกราฟอะไร

**เฉลย:**
- จัดกลุ่ม: $4(x^2 - 4x) + 9(y^2 + 2y) = 11$
- กำลังสองสมบูรณ์: $4(x-2)^2 - 16 + 9(y+1)^2 - 9 = 11$
- $4(x-2)^2 + 9(y+1)^2 = 36$
- หาร 36: $\dfrac{(x-2)^2}{9} + \dfrac{(y+1)^2}{4} = 1$

**ตอบ:** วงรี center $(2, -1)$, $a = 3$ (แกนเอกขนาน $X$), $b = 2$
