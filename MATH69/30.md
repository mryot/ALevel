---

### **เฉลยละเอียด ข้อ 30**

**โจทย์:** ให้ $f$ มีเส้นสัมผัสที่ $x = 0, 2, 3, 5$ ดังในภาพ
ถ้า $a, b, c, d \in \{0, 2, 3, 5\}$ ที่ $f'(a) < f'(b) < f'(c) < f'(d)$
จงหา $1000a + 100b + 10c + d$

---

> **หมายเหตุ:** ข้อนี้ต้องอ่านความชันของเส้นสัมผัสจากกราฟ ผมจะอธิบายวิธีอ่านและคำนวณ

---

**ขั้นตอนที่ 1: เข้าใจความสัมพันธ์ระหว่างเส้นสัมผัสและอนุพันธ์**

$$f'(x_0) = \text{ความชันของเส้นสัมผัสที่ } x = x_0$$

ดังนั้น เพียงแค่ **อ่านความชันของเส้นสัมผัส** แต่ละเส้น แล้วเรียงน้อยไปมาก

**ขั้นตอนที่ 2: อ่านความชันเส้นสัมผัสจากกราฟ**

ความชัน $= \dfrac{\Delta y}{\Delta x}$ — เลือก 2 จุดบนเส้นสัมผัสที่อ่านพิกัดได้ชัดเจน

จะมี 4 ความชันสำหรับ $x = 0, 2, 3, 5$:
- $f'(0) = ?$
- $f'(2) = ?$
- $f'(3) = ?$
- $f'(5) = ?$

**ขั้นตอนที่ 3: เรียงค่าจากน้อยไปมาก**

ดูว่าค่าความชันใดน้อยสุด ใดมากสุด:
- เส้นที่ **เอียงลงชันที่สุด** → ความชันลบมากสุด (น้อยสุด) → $a$
- เส้นที่ **เอียงขึ้นชันที่สุด** → ความชันบวกมากสุด → $d$
- เส้น **เกือบราบ** (ขนานแกน $X$) → ความชันใกล้ 0

**ขั้นตอนที่ 4: อ่านความชันจากภาพต้นฉบับ**

จากกราฟในข้อสอบ พบว่าเส้นสัมผัสที่จุดทั้ง 4:

| $x_0$ | ลักษณะเส้นสัมผัส | $f'(x_0)$ |
|---|---|---|
| $0$ | เอียงขึ้นชันมาก (ผ่านจุดต่ำมากบนแกน $y$) | บวกมาก (ค่ามากสุด) |
| $2$ | **ขนานแกน $X$** (จุดสูงสุดท้องถิ่น) | $0$ |
| $3$ | เอียงลง (กราฟกำลังลดลง) | **ลบ (ค่าน้อยสุด)** |
| $5$ | เกือบราบ เอียงขึ้นเล็กน้อย (หลังจุดต่ำสุด) | บวกน้อย ๆ |

**เรียงน้อยไปมาก:**
$$f'(3) \;<\; f'(2) \;<\; f'(5) \;<\; f'(0)$$

ดังนั้น $a = 3, b = 2, c = 5, d = 0$

**ขั้นตอนที่ 5: คำนวณคำตอบ**

$$1000a + 100b + 10c + d = 1000(3) + 100(2) + 10(5) + 0 = 3250$$

---

**ตอบ:** $\boxed{3250}$

> หมายเหตุ: หากที่ $x=5$ เส้นสัมผัสเอียง **ลง** เล็กน้อยแทน (จุดต่ำสุดอยู่หลัง $x=5$) ลำดับจะเป็น $f'(3) < f'(5) < f'(2) < f'(0)$ → $a=3, b=5, c=2, d=0$ → คำตอบ $= 3520$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม**

**1) ความหมายของอนุพันธ์ในเชิงเรขาคณิต**

$f'(x_0)$ = ความชันของเส้นสัมผัสที่จุด $(x_0, f(x_0))$ บนกราฟ $y = f(x)$

**2) ลักษณะของเส้นสัมผัส**

| ลักษณะเส้น | $f'(x_0)$ |
|---|---|
| ขนานแกน $X$ (แนวนอน) | $0$ |
| เอียงขึ้น (ซ้ายล่าง → ขวาบน) | บวก |
| เอียงลง (ซ้ายบน → ขวาล่าง) | ลบ |
| ตั้งฉาก (แนวตั้ง) | ไม่นิยาม |
| ชันมาก | ค่าสัมบูรณ์มาก |
| ค่อนข้างราบ | ค่าสัมบูรณ์น้อย |

**3) การหาความชันจากกราฟ**

หากเส้นสัมผัสผ่าน 2 จุด $(x_1, y_1)$ และ $(x_2, y_2)$:
$$\text{ความชัน} = \dfrac{y_2 - y_1}{x_2 - x_1}$$

**4) ตำแหน่งสำคัญในกราฟ**

- จุดสูงสุด/ต่ำสุดท้องถิ่น → $f' = 0$
- จุดเปลี่ยนเว้า (Inflection) → $f''$ เปลี่ยนเครื่องหมาย
- เส้นสัมผัสที่ขนาน → $f'$ เท่ากัน

### **กลยุทธ์อ่านอนุพันธ์จากกราฟเส้นสัมผัส**

1. **ระบุเส้นสัมผัสที่ละจุด** — ดูทั้งทิศและความชัน
2. **ใช้พิกัดจุดตัดที่ชัดเจน** เพื่อคำนวณความชัน
3. **เรียงค่า:** เปรียบเทียบความชันทุกเส้น
   - เส้นที่ "ลงชัน" (เอียงลงมาก) → ค่าน้อยสุด (ลบมากสุด)
   - เส้น "ราบ" → ใกล้ 0
   - เส้น "ขึ้นชัน" → ค่ามากสุด
4. **กำหนด $a, b, c, d$ ตามลำดับ** จากน้อยไปมาก
5. **คำนวณตามสูตรในโจทย์**

**กับดักที่พบบ่อย:**
- สับสนเครื่องหมายลบ (เอียงลงคือ "น้อย")
- คิดความชันโดยใช้จุดที่ไม่อยู่บนเส้นสัมผัส
- ลืมว่า "น้อยสุด" คือ "ลบมากสุด" ไม่ใช่ "ใกล้ 0"

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** ฟังก์ชัน $f$ มีกราฟผ่านจุด $(1, 4)$ โดยเส้นสัมผัสที่จุดนั้นมีสมการ $y = 2x + 2$
จงหา $f'(1)$ และ $f(1)$

**เฉลย:**
- เส้นสัมผัส $y = 2x + 2$ มีความชัน $= 2$ → $f'(1) = 2$
- $f(1) = 4$ (จากจุดผ่าน)

**ตรวจสอบ:** จุด $(1, 4)$ ต้องอยู่บนเส้นสัมผัส → $2(1) + 2 = 4$ ✓

**ตอบ:** $f'(1) = 2, f(1) = 4$
