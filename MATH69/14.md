### **เฉลยละเอียด ข้อ 14**

**โจทย์:** จงหา
$$\sum_{n=1}^{\infty} \left(\dfrac{1}{3}\right)^n \sin\!\left(\dfrac{n\pi}{2}\right)$$

---

**ขั้นตอนที่ 1: หาค่า $\sin(n\pi/2)$ สำหรับ $n = 1, 2, 3, 4, \ldots$**

| $n$ | $n\pi/2$ | $\sin(n\pi/2)$ |
|---|---|---|
| 1 | $\pi/2$ | $1$ |
| 2 | $\pi$ | $0$ |
| 3 | $3\pi/2$ | $-1$ |
| 4 | $2\pi$ | $0$ |
| 5 | $5\pi/2$ | $1$ |
| 6 | $3\pi$ | $0$ |
| 7 | $7\pi/2$ | $-1$ |

**รูปแบบ:** $1, 0, -1, 0, 1, 0, -1, 0, \ldots$ — พจน์เลขคู่เป็น 0, พจน์เลขคี่สลับ $\pm 1$

**ขั้นตอนที่ 2: เขียนผลบวกแบบเก็บเฉพาะพจน์ที่ไม่เป็น 0**

$$S = \dfrac{1}{3} \cdot 1 + \dfrac{1}{3^3} \cdot (-1) + \dfrac{1}{3^5} \cdot 1 + \dfrac{1}{3^7} \cdot (-1) + \cdots$$

$$= \dfrac{1}{3} - \dfrac{1}{3^3} + \dfrac{1}{3^5} - \dfrac{1}{3^7} + \cdots$$

**ขั้นตอนที่ 3: เป็นอนุกรมเรขาคณิตอนันต์**

พจน์แรก $a = \dfrac{1}{3}$
อัตราส่วนร่วม: เปลี่ยน $3 \to 3^3$ และเปลี่ยนเครื่องหมาย → $r = -\dfrac{1}{3^2} = -\dfrac{1}{9}$

ตรวจสอบ: $|r| = 1/9 < 1$ → ลู่เข้า

$$S = \dfrac{a}{1 - r} = \dfrac{1/3}{1 - (-1/9)} = \dfrac{1/3}{10/9} = \dfrac{1}{3} \cdot \dfrac{9}{10} = \dfrac{3}{10}$$

**ตอบ:** ข้อ 2. $\dfrac{3}{10}$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม**

**1) อนุกรมเรขาคณิต**

| ประเภท | สูตร | เงื่อนไข |
|---|---|---|
| จำกัด $n$ พจน์ | $S_n = \dfrac{a(1 - r^n)}{1 - r}$ | $r \ne 1$ |
| อนันต์ | $S_\infty = \dfrac{a}{1 - r}$ | $\|r\| < 1$ |

**2) ค่า $\sin(n\pi/2)$**

วงรอบ 4: $\sin(n\pi/2) = 1, 0, -1, 0, 1, 0, -1, 0, \ldots$ สำหรับ $n=1,2,3,\ldots$

**3) การจัดการอนุกรมที่มี 0 เป็นพจน์**

- ตัดพจน์ที่เป็น 0 ออก
- พจน์ที่เหลือมักกลายเป็นอนุกรมเรขาคณิตใหม่
- เปลี่ยนตัวแปร เช่น ให้พจน์คี่ $n = 2k-1$

---

### **กลยุทธ์แก้โจทย์อนุกรม sin/cos × geometric**

1. **เขียนค่า trig ของแต่ละ $n$** ใน 4 พจน์แรก (วงรอบ)
2. **เก็บเฉพาะพจน์ที่ไม่เป็น 0** → จะได้อนุกรมที่ง่ายขึ้น
3. **เช็คว่าเป็นอนุกรมเรขาคณิต** หรือไม่
4. **หา $a$ และ $r$** ของอนุกรมใหม่
5. **ตรวจ $|r| < 1$** ก่อนใช้สูตรอนันต์
6. **อีกวิธี:** ใช้สูตรของออยเลอร์ $\sin(n\pi/2) = \mathrm{Im}(i^n)$ ผสมกับ geometric

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** จงหาผลบวกของ
$$\sum_{n=1}^{\infty}\left(\dfrac{1}{2}\right)^n \cos\!\left(\dfrac{n\pi}{2}\right)$$

**เฉลย:**
$\cos(n\pi/2)$: สำหรับ $n=1,2,3,4,\ldots$ ได้ $0, -1, 0, 1, 0, -1, 0, 1, \ldots$

พจน์ที่ไม่เป็น 0: เริ่ม $n = 2$
$$S = -\dfrac{1}{4} + \dfrac{1}{16} - \dfrac{1}{64} + \cdots$$

$a = -1/4$, $r = -1/4$
$$S = \dfrac{-1/4}{1 - (-1/4)} = \dfrac{-1/4}{5/4} = -\dfrac{1}{5}$$

**ตอบ:** $-\dfrac{1}{5}$
