### **เฉลยละเอียด ข้อ 22**

**โจทย์:** อายุหลอดไฟ ~ Normal($\mu = 2, \sigma = 0.5$) ปี
จงหา $P(X > 2.5)$

---

**ขั้นตอนที่ 1: แปลงเป็นมาตรฐาน (Standardization)**

$$z = \dfrac{x - \mu}{\sigma} = \dfrac{2.5 - 2}{0.5} = \dfrac{0.5}{0.5} = 1$$

**ขั้นตอนที่ 2: คำนวณความน่าจะเป็น**

$$P(X > 2.5) = P(Z > 1) = 1 - P(Z \le 1)$$

จากตาราง: $P(Z \le 1) = 0.8413$

$$P(X > 2.5) = 1 - 0.8413 = 0.1587$$

**ตอบ:** ข้อ 2. $0.1587$

---

### **เนื้อหาเพื่อศึกษาเพิ่มเติม: การแจกแจงปกติ (Normal Distribution)**

**1) คุณสมบัติของการแจกแจงปกติ**

ตัวแปรสุ่ม $X \sim N(\mu, \sigma^2)$ มีกราฟ "ระฆังคว่ำ":
- สมมาตรรอบ $\mu$
- พื้นที่ใต้กราฟทั้งหมด $= 1$
- $\sigma$ ใหญ่ → กระจายมาก, $\sigma$ น้อย → แคบ

**2) คะแนนมาตรฐาน (z-score)**

$$z = \dfrac{x - \mu}{\sigma}$$

แปลง $X \sim N(\mu, \sigma^2)$ ให้เป็น $Z \sim N(0, 1)$ (การแจกแจงปกติมาตรฐาน)

**3) สูตรพื้นฐานในการเปิดตาราง**

| ที่ต้องการ | สูตร |
|---|---|
| $P(X \le a)$ | $P(Z \le z_a)$ |
| $P(X > a)$ | $1 - P(Z \le z_a)$ |
| $P(a \le X \le b)$ | $P(Z \le z_b) - P(Z \le z_a)$ |
| $P(X < -a)$ ใช้สมมาตร | $P(Z \le -z_a) = 1 - P(Z \le z_a)$ |

**4) กฎ 68-95-99.7**

- $\mu \pm 1\sigma$ ครอบคลุม ~68%
- $\mu \pm 2\sigma$ ครอบคลุม ~95%
- $\mu \pm 3\sigma$ ครอบคลุม ~99.7%

---

### **กลยุทธ์แก้โจทย์การแจกแจงปกติ**

1. **ระบุ $\mu, \sigma$ จากโจทย์ให้ชัด**
2. **แปลงเป็น z-score** ทุกครั้งก่อนเปิดตาราง
3. **เขียนเงื่อนไขในรูป $P(Z \le z)$** ก่อน เพื่อใช้กับตารางได้ตรง
4. **ใช้สมมาตร:** $P(Z \le -z) = 1 - P(Z \le z)$
5. **ตรวจสอบความสมเหตุสมผล:** ค่า P อยู่ใน $[0, 1]$

**กับดักที่พบบ่อย:**
- ลืมหาร $\sigma$ ตอนคำนวณ z-score
- ใช้ตาราง $P(Z \le z)$ แทน $P(Z > z)$ โดยไม่ลบ 1
- สับสนระหว่าง $\sigma$ (SD) และ $\sigma^2$ (variance)

---

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** น้ำหนักนักเรียนกระจายปกติ $\mu = 60$ กก., $\sigma = 8$ กก.
จงหาความน่าจะเป็นที่นักเรียนคนหนึ่งหนักระหว่าง 52 ถึง 68 กก.

**เฉลย:**
- $z_1 = (52-60)/8 = -1$
- $z_2 = (68-60)/8 = 1$
- $P(52 \le X \le 68) = P(-1 \le Z \le 1) = P(Z \le 1) - P(Z \le -1)$
- $= 0.8413 - 0.1587 = 0.6826$ (ประมาณ 68%)

**ตอบ:** ประมาณ $0.6826$ (สอดคล้องกับกฎ 68%)
