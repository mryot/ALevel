### **เฉลยละเอียด ข้อ 7**

**โจทย์:** ให้ $S$ เป็นเซตคำตอบที่เป็นจำนวนจริงทั้งหมดของสมการ
$$\dfrac{2^{x^2}}{100} = \dfrac{100}{25^x}$$
หาผลบวกของสมาชิกในเซต $S$

---

**ขั้นตอนที่ 1: จัดรูปสมการ**

คูณไขว้:
$$2^{x^2} \cdot 25^x = 100 \cdot 100 = 10000 = 10^4$$

**ขั้นตอนที่ 2: ใส่ $\log_2$ ทั้งสองข้าง**

$$\log_2\left(2^{x^2} \cdot 25^x\right) = \log_2(10^4)$$
$$x^2 + x \log_2 25 = 4\log_2 10$$
$$x^2 + 2x \log_2 5 = 4(\log_2 2 + \log_2 5)$$
$$x^2 + 2x \log_2 5 = 4 + 4\log_2 5$$

**ขั้นตอนที่ 3: ตั้งเป็นสมการกำลังสอง**

ให้ $a = \log_2 5$:
$$x^2 + 2ax - 4 - 4a = 0$$
$$x^2 + 2ax - 4(1 + a) = 0$$

**ขั้นตอนที่ 4: ใช้สูตร Vieta หาผลบวกของราก**

สำหรับสมการ $x^2 + 2ax - 4(1+a) = 0$:
$$x_1 + x_2 = -\dfrac{2a}{1} = -2a = -2\log_2 5$$

**ตรวจสอบโดยแยกตัวประกอบ:**
$$x^2 + 2ax - 4(1+a) = (x - 2)(x + 2 + 2a) = 0$$
ขยาย: $x^2 + (2+2a)x - 2x - 4 - 4a = x^2 + 2ax - 4 - 4a$ ✓

ราก: $x_1 = 2$, $x_2 = -2 - 2a = -2 - 2\log_2 5 = -2\log_2(10) = -\log_2 100$

ผลบวก: $2 + (-2 - 2\log_2 5) = -2\log_2 5$

---

**ตอบ:** ข้อ 4. $-2\log_2 5$

### **เนื้อหาเพื่อศึกษาเพิ่มเติม: สมการเอ็กซ์โพเนนเชียล-ลอการิทึม**

**1) สมบัติพื้นฐาน**

| สูตร | ตัวอย่าง |
|---|---|
| $a^{m+n} = a^m \cdot a^n$ | $2^5 = 2^2 \cdot 2^3$ |
| $(a^m)^n = a^{mn}$ | $(2^3)^4 = 2^{12}$ |
| $\log(MN) = \log M + \log N$ | $\log 6 = \log 2 + \log 3$ |
| $\log M^n = n \log M$ | $\log 8 = 3 \log 2$ |
| $\log_a b = \dfrac{\log b}{\log a}$ | $\log_2 8 = \dfrac{\log 8}{\log 2} = 3$ |

**2) วิธีแก้สมการเอ็กซ์โพเนนเชียล**

| รูปสมการ | วิธีแก้ |
|---|---|
| $a^{f(x)} = a^{g(x)}$ | เทียบเลขชี้กำลัง: $f(x) = g(x)$ |
| $a^{f(x)} = b^{g(x)}$ | ใส่ log สองข้าง |
| $a^{2x} - 3a^x + 2 = 0$ | แทน $t = a^x$ แล้วแก้กำลังสอง |

**3) สูตร Vieta สำหรับผลบวก-ผลคูณราก**

สมการ $ax^2 + bx + c = 0$ มีราก $x_1, x_2$:
- ผลบวก: $x_1 + x_2 = -\dfrac{b}{a}$
- ผลคูณ: $x_1 \cdot x_2 = \dfrac{c}{a}$

### **กลยุทธ์แก้โจทย์ exp/log**

1. **เทียบฐานเดียวกัน** ถ้าทำได้ (ลด exponent ลงตัว)
2. **ถ้าฐานต่างกัน:** ใส่ log ทั้งสองข้าง
3. **เก็บค่าในรูป $\log$** ไม่ต้องเปลี่ยนเป็นทศนิยม — ตัวเลือกมักอยู่ในรูปสัญลักษณ์
4. **ระวังโดเมน:** ในสมการ log ต้องมี argument > 0
5. **ตรวจคำตอบ:** แทนกลับเข้าสมการเดิม

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** จงหาผลบวกของคำตอบของสมการ $3^{x^2} = 9^{x+1}$

**เฉลย:**
$3^{x^2} = 3^{2(x+1)} = 3^{2x+2}$
เทียบเลขชี้กำลัง: $x^2 = 2x + 2$
$x^2 - 2x - 2 = 0$
ผลบวกของราก (Vieta) = $-(-2)/1 = 2$

**ตอบ:** 2
