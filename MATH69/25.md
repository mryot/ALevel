### **เฉลยละเอียด ข้อ 25**

**โจทย์:** อัตราการเปลี่ยนแปลงของค่าใช้จ่าย $f(x) = 8x + 6$ (พันบาท/วัน)
ค่าใช้จ่ายรวมจากวันที่ 5 ถึงวันที่ 10 เท่ากับเท่าใด

---

**ขั้นตอนที่ 1: เข้าใจความหมาย**

"อัตราการเปลี่ยนแปลงของค่าใช้จ่าย" = อนุพันธ์ของค่าใช้จ่ายสะสม
→ ค่าใช้จ่ายในช่วงเวลาหนึ่ง = **อินทิกรัล** ของอัตรา

$$\text{ค่าใช้จ่ายรวม} = \int_a^b f(x)\,dx$$

**ขั้นตอนที่ 2: ตั้งสูตรอินทิกรัล**

$$\text{ค่าใช้จ่ายจากวันที่ 5 ถึง 10} = \int_5^{10}(8x + 6)\,dx$$

**ขั้นตอนที่ 3: หาปฏิยานุพันธ์**

$$\int (8x + 6)\,dx = 4x^2 + 6x + C$$

**ขั้นตอนที่ 4: ประเมินค่าตามขอบเขต**

$$\int_5^{10}(8x+6)\,dx = \left[4x^2 + 6x\right]_5^{10}$$

ที่ $x = 10$: $4(100) + 60 = 460$
ที่ $x = 5$: $4(25) + 30 = 130$

$$= 460 - 130 = 330 \text{ (พันบาท)}$$

**ขั้นตอนที่ 5: แปลงหน่วย**

$330$ พันบาท $= 330{,}000$ บาท

**ตอบ:** ข้อ 3. $330{,}000$ บาท

### **เนื้อหาเพื่อศึกษาเพิ่มเติม**

**1) ทฤษฎีบทพื้นฐานของแคลคูลัส (FTC)**

ถ้า $F$ เป็นปฏิยานุพันธ์ของ $f$ แล้ว:
$$\int_a^b f(x)\,dx = F(b) - F(a)$$

**2) ความหมายของอัตราเทียบกับปริมาณรวม**

| ปริมาณ | อัตราการเปลี่ยน | ปริมาณรวมในช่วง |
|---|---|---|
| ระยะทาง | ความเร็ว $v(t)$ | $\int v\,dt$ |
| ค่าใช้จ่าย | อัตราค่าใช้จ่าย/เวลา | $\int f\,dx$ |
| ปริมาณน้ำ | อัตราการไหล | $\int Q\,dt$ |
| ประชากร | อัตราการเติบโต | $\int r\,dt$ |

**3) สูตรอินทิกรัลพื้นฐาน**

| สูตร |
|---|
| $\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C$, $n \ne -1$ |
| $\int e^x\,dx = e^x + C$ |
| $\int \dfrac{1}{x}\,dx = \ln|x| + C$ |
| $\int \sin x\,dx = -\cos x + C$ |
| $\int \cos x\,dx = \sin x + C$ |

### **กลยุทธ์แก้โจทย์ประยุกต์ปริพันธ์**

1. **ระบุ "อัตรา" vs "ปริมาณรวม"** → ตัดสินใจว่าใช้อนุพันธ์หรืออินทิกรัล
2. **ตั้งขอบเขต $a$ ถึง $b$ ให้ตรงโจทย์** (วันที่เริ่ม → วันที่สิ้นสุด)
3. **หาปฏิยานุพันธ์ก่อน** จากนั้นค่อยแทนขอบเขต
4. **ตรวจหน่วย:** หน่วยของอัตรา × หน่วยของตัวแปร = หน่วยปริมาณรวม
5. **แปลงหน่วยตามโจทย์** (เช่น พันบาท → บาท)

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** รถยนต์ออกเดินทางด้วยความเร็ว $v(t) = 3t^2 + 2t$ (เมตร/วินาที)
จงหาระยะทางที่รถวิ่งได้ในช่วงวินาทีที่ 2 ถึงวินาทีที่ 5

**เฉลย:**
$$s = \int_2^5 (3t^2 + 2t)\,dt = \left[t^3 + t^2\right]_2^5$$

ที่ $t=5$: $125 + 25 = 150$
ที่ $t=2$: $8 + 4 = 12$

$s = 150 - 12 = 138$ เมตร

**ตอบ:** $138$ เมตร
