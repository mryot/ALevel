### **เฉลยละเอียด ข้อ 24**

**โจทย์:** จากกราฟของ $f$ ข้อใดถูกต้อง
- 1. $\displaystyle\lim_{x\to 2^+}f(x) < \lim_{x\to 2^-}f(x) < f(2)$
- 2. $\displaystyle\lim_{x\to 2^+}f(x) < \lim_{x\to 2^-}f(x) = f(2)$
- 3. $\displaystyle\lim_{x\to 2^+}f(x) = f(2) < \lim_{x\to 2^-}f(x)$
- 4. $\displaystyle\lim_{x\to 2^-}f(x) < \lim_{x\to 2^+}f(x) = f(2)$
- 5. $\displaystyle\lim_{x\to 2^-}f(x) = f(2) < \lim_{x\to 2^+}f(x)$

---

> **หมายเหตุ:** ข้อนี้ต้องอ่านค่าจากกราฟ ผมจะอธิบายวิธีอ่านและตีความแต่ละสัญลักษณ์

---

**วิธีอ่านกราฟที่ $x = 2$**

จะมี **3 ค่า** ที่ต้องอ่านอย่างระวัง:
1. **ลิมิตซ้าย** $\displaystyle\lim_{x\to 2^-}f(x)$ : ค่า $y$ ที่กราฟ "เข้าหา" เมื่อ $x$ เข้าใกล้ $2$ จากซ้าย (ปลายลูกศรชี้)
2. **ลิมิตขวา** $\displaystyle\lim_{x\to 2^+}f(x)$ : ค่า $y$ ที่กราฟ "เข้าหา" เมื่อ $x$ เข้าใกล้ $2$ จากขวา
3. **ค่าของฟังก์ชัน** $f(2)$ : ค่า $y$ ที่กราฟ "จุดทึบ" (●) วางอยู่ ณ $x = 2$

**สัญลักษณ์บนกราฟ:**
- จุด **เปิด** (○) = ค่านั้นไม่ใช่ค่าของฟังก์ชัน (ใช้แสดงลิมิต)
- จุด **ทึบ** (●) = ค่าของฟังก์ชัน $f(2)$

---

**วิธีอ่านจากกราฟต้นฉบับ:**

ที่ $x = 2$ พบว่า:
- เส้นกราฟจาก **ซ้าย** ไต่ขึ้นเข้าหาค่าสูง → $\displaystyle\lim_{x\to 2^-}f(x) = 4$ (มีจุดทึบ ● ที่ $(2, 4)$)
- มี **จุดทึบ** (●) ที่ $(2, 4)$ → $f(2) = 4$
- มี **จุดเปิด** (○) ที่ $(2, 0)$ และเส้นกราฟด้าน **ขวา** ออกจากจุดนี้ → $\displaystyle\lim_{x\to 2^+}f(x) = 0$

**เปรียบเทียบค่า:**
$$\lim_{x\to 2^+}f(x) = 0 \;<\; \lim_{x\to 2^-}f(x) = f(2) = 4$$

ตรงกับ **ข้อ 2** : $\displaystyle\lim_{x\to 2^+}f(x) < \lim_{x\to 2^-}f(x) = f(2)$

---

**ตอบ:** ข้อ 2. $\displaystyle\lim_{x\to 2^+}f(x) < \lim_{x\to 2^-}f(x) = f(2)$

### **เนื้อหาเพื่อศึกษาเพิ่มเติม: ลิมิตและความต่อเนื่อง**

**1) นิยามลิมิต**

$$\lim_{x \to a} f(x) = L$$

หมายถึง เมื่อ $x$ เข้าใกล้ $a$ (จากทั้งซ้ายและขวา) ค่าของ $f(x)$ จะเข้าใกล้ $L$

**2) ลิมิตซ้าย-ขวา**

- ลิมิตซ้าย: $\displaystyle\lim_{x \to a^-} f(x)$
- ลิมิตขวา: $\displaystyle\lim_{x \to a^+} f(x)$
- ลิมิตมีอยู่ ⟺ ลิมิตซ้าย = ลิมิตขวา

**3) ความต่อเนื่องที่จุด $x = a$**

$f$ **ต่อเนื่อง** ที่ $x = a$ ⟺ ทั้ง 3 เงื่อนไขเป็นจริง:
1. $f(a)$ มีค่า
2. $\displaystyle\lim_{x\to a} f(x)$ มีค่า
3. $\displaystyle\lim_{x\to a} f(x) = f(a)$

**4) ประเภทของความไม่ต่อเนื่อง**

| ประเภท | ลักษณะ |
|---|---|
| Removable | ลิมิตมี แต่ ≠ $f(a)$ (หรือ $f(a)$ ไม่นิยาม) |
| Jump | ลิมิตซ้าย ≠ ลิมิตขวา |
| Infinite | ลิมิตเข้าสู่ $\pm\infty$ |

---

### **กลยุทธ์อ่านกราฟลิมิต**

1. **ระบุจุดที่สนใจ** $x = a$
2. **อ่านลิมิตซ้าย:** มองเส้นกราฟด้านซ้าย ติดตามทิศที่เข้าหา
3. **อ่านลิมิตขวา:** มองเส้นกราฟด้านขวา ติดตามทิศที่เข้าหา
4. **หาจุดทึบ** เพื่อระบุ $f(a)$
5. **เปรียบเทียบ 3 ค่า:** ลิมิตซ้าย, ลิมิตขวา, $f(a)$ — ใครเท่าใคร, ใครมากกว่า
6. **เช็คตัวเลือก:** เลือกข้อที่มีลำดับและความสัมพันธ์ตรงกัน

---

### **ตัวอย่างโจทย์เพิ่มเติม**

**โจทย์:** กำหนด $f(x) = \begin{cases} x + 1 & \text{ถ้า } x < 2 \\ 5 & \text{ถ้า } x = 2 \\ x^2 & \text{ถ้า } x > 2 \end{cases}$

จงหา $\displaystyle\lim_{x\to 2^-}f(x), \lim_{x\to 2^+}f(x), f(2)$ และเรียงลำดับ

**เฉลย:**
- $\lim_{x\to 2^-}f(x) = 2 + 1 = 3$
- $\lim_{x\to 2^+}f(x) = 2^2 = 4$
- $f(2) = 5$

**ลำดับ:** $\lim_{x\to 2^-}f(x) < \lim_{x\to 2^+}f(x) < f(2)$ → ฟังก์ชันไม่ต่อเนื่องที่ $x = 2$
