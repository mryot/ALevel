# สรุปสั้น Math66 สำหรับทบทวนเร็ว

เอกสารนี้ย่อแนวคิดสำคัญของ MATH66 ให้เหลือเฉพาะสิ่งที่ควรนึกให้ได้ทันทีตอนทำข้อสอบ

## 1) สูตรที่ต้องจำทันที

- เซต: n(A∪B)=n(A)+n(B)-n(A∩B)
- De Morgan: (A∪B)'=A'∩B', (A∩B)'=A'∪B'
- พหุนาม: ทฤษฎีบทเศษเหลือ, ทฤษฎีบทตัวประกอบ
- ตรีโกณ: กฎไซน์, กฎโคไซน์, สูตรมุมสองเท่า
- ลำดับ: a_n=a_1+(n-1)d
- อนุกรมเรขาคณิตอนันต์: S_∞=a/(1-r), |r|<1
- เมทริกซ์: det ใช้หาค่าตัวแปรและตรวจความสัมพันธ์
- เวกเตอร์: u·v=|u||v|cosθ, |u·(v×w)| คือปริมาตร
- binomial: P(X=k)=C(n,k)p^k(1-p)^(n-k)
- normal: z=(x-μ)/σ
- แคลคูลัส: lim, f'(x), พื้นที่ใต้กราฟ

## 2) แนวคิดย่อรายหมวด

### เซตและตรรกศาสตร์
- แปลคำว่า ทุก, บาง, ถ้า...แล้ว, ก็ต่อเมื่อ ให้เป็นสัญลักษณ์ก่อน
- ข้อที่มี "จริงกี่ข้อ" ให้เช็กทีละประพจน์

### พีชคณิตและฟังก์ชัน
- ถ้าเห็น log ต้องเช็กโดเมนก่อน
- ถ้าเห็นฟังก์ชันผกผัน ต้องดูว่าเป็น one-to-one หรือไม่
- ถ้าเจอจำนวนเชิงซ้อน ให้มองรูปเชิงขั้วเมื่อมีการยกกำลังหรือหาราก

### เรขาคณิตและเวกเตอร์
- ภาคตัดกรวยให้เขียนรูปมาตรฐานก่อน
- เวกเตอร์สามมิติมักจบที่ dot/cross/determinant
- จุดศูนย์กลางกับเส้นสัมผัสใช้ระยะและความชันช่วยได้

### สถิติและความน่าจะเป็น
- คำว่า อย่างน้อย ใช้ทางตรงข้ามบ่อยครั้งจะเร็วกว่า
- binomial ต้องรู้ว่าเหตุการณ์มี 2 ผลลัพธ์และอิสระต่อกัน
- normal distribution ให้เปลี่ยนเป็น z-score ก่อนอ่านพื้นที่

### แคลคูลัส
- เริ่มจากลิมิตหรือรูปกราฟก่อน ไม่ใช่กระโดดเข้าดิฟทันที
- ข้อพื้นที่ใต้กราฟ ถ้ากราฟอยู่ใต้แกน x ต้องกลับเครื่องหมายให้เป็นพื้นที่บวก

## 3) ลำดับการทำข้อสอบแบบเร็ว

1. เก็บข้อที่สูตรชัดก่อน
2. ข้อกราฟหรือแผนภาพอ่านข้อมูลจากรูปก่อน
3. ข้อที่มีหลายเงื่อนไขให้ตั้งสมการทีละก้อน
4. ข้อที่ยังไม่จบใน 2-3 นาทีให้ข้ามก่อน

## 4) จุดผิดพลาดที่เจอบ่อย

- อ่านคำถามผิดว่าให้หาค่าอะไร
- ลืมเงื่อนไขโดเมนของ log และฟังก์ชัน
- นับซ้ำในโจทย์การนับ
- ใช้ช่วงคำตอบตรีโกณไม่ครบ
- อ่านกราฟผิดทิศทาง

## 5) สูตรจำสั้นๆ ตอนก่อนเข้าห้องสอบ

- เซต = รวม-ตัด-แยก
- ตรรกศาสตร์ = ตารางจริงและ De Morgan
- ฟังก์ชัน = โดเมนก่อนเสมอ
- เวกเตอร์ = dot, cross, determinant
- ความน่าจะเป็น = complement และ binomial
- สถิติ = z-score และ box plot
- แคลคูลัส = lim → derivative → area
